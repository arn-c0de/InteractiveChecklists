About the Project

InteractiveChecklists is a modern Android application built with Jetpack Compose, designed for viewing and interacting with Markdown and PDF checklists. Perfect for aviation, military simulations, and professional workflows.

📱

Android Native

Built with modern Jetpack Compose for smooth performance

📝

Markdown Support

Full Markdown rendering with interactive elements

🔒

Secure

ECDH encryption for secure data transfer

Key Features

  • 📄 View and interact with Markdown and PDF checklists
  • 🗺️ Interactive tactical maps with military symbols
  • ✈️ Flight instruments and traffic pattern visualization
  • 📊 Tactical units tracking and route planning
  • 🏷️ Advanced tag system for organization
  • 📝 Quick notes with DataPad feature
  • 🔐 Secure device-to-device communication
  • 🎨 Modern Material Design 3 interface
