# A-10C Warthog Taxi Checklist

## Before Taxi

- [ ] **APU generator switch** - OFF
- [ ] **APU switch** - OFF
- [ ] **Parking brake** - As required
- [ ] **Nose wheel steering** - Check
- [ ] **Brakes** - Check
- [ ] **Flight instruments** - Check
- [ ] **Attitude indicator** - Check
- [ ] **HSI/ADI** - Check
- [ ] **Heading** - Check with compass

## Taxi Procedures

- [ ] **Check flight control movements** - Monitor Hydraulic System Pressure Gauge (2,800psi - 3,350psi)
- [ ] **HOTAS Trim switch** - Test
- [ ] **PITCH/ROLL TRIM switch** - Set NORM EMER OVERRIDE
- [ ] **EMERGENCY PITCH & ROLL switch** - Test
- [ ] **Manual trim check** - PITCH/ROLL TRIM switch
- [ ] **TRIM indicator** - Check centered
- [ ] **Flight controls** - Full deflection check
- [ ] **Aileron** - Check movement
- [ ] **Elevator** - Check movement
- [ ] **Rudder** - Check movement
- [ ] **Speed brake** - Extend/retract, check operation
- [ ] **Flaps** - Set to takeoff position
- [ ] **Taxi/Landing lights** - As required
- [ ] **NVIS lights** - As required
- [ ] **Pitot heat** - ON
- [ ] **Engine anti-ice** - As required
- [ ] **Wing/tail anti-ice** - As required
- [ ] **Transponder** - ALT
- [ ] **IFF** - Set as required
