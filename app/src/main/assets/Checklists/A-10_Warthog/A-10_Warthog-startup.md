# A-10C Warthog Startup Checklist

## Prior to Engine Start

- [ ] **BAT switch (BAT)** - PWR
- [ ] **UHF controls** – ON/Set
- [ ] **Inverter switch** - STBY
- [ ] **Engine instruments** - Check, ITT indicator below 150°C, OFF flag
- [ ] **Fire detect/bleed air leak test button** - Depress
- [ ] **Gear lights** - Check (three green, no red)
- [ ] **Signal lights** - Test
- [ ] **Clock** - Set
- [ ] **Fuel quantity** - Check
- [ ] **Oxygen quantity** - Check. SUPPLY lever - ON
- [ ] **APU switch** - START (or signal for external air)
- [ ] **L-FUEL PRESS light** - Check off
- [ ] **APU generator switch** - PWR
- [ ] **AAP: CDU switch** - ON
- [ ] **CDU** - Set for desired alignment
- [ ] **VHF/AM** - TR. Frequency - verified, Set as req
- [ ] **VHF/FM** - TR
- [ ] **Seat** - Adjust
- [ ] **Armament/ground safety override switch** - SAFE (guard down)
- [ ] **IFF antenna** - BOTH
- [ ] **CCTVS/DVADR remote control panel (left canopy bow)** - OFF
- [ ] **KY-58** – OFF/Set
- [ ] **Intercom controls** - Set
- [ ] **Radar altimeter** - NRM
- [ ] **Emergency flight controls** - Set
- [ ] **HARS/SAS override switch** - NORM
- [ ] **Refuel status and indexer light control** - Set
- [ ] **NVIS lights switch** - OFF
- [ ] **Throttle master exterior light control switch** - AFT
- [ ] **Flap lever** - Set to flap position
- [ ] **Landing gear handle** - DOWN
- [ ] **Landing/taxi light switch** - OFF
- [ ] **Armament HUD control panel** - Set: MASTER ARM switch - SAFE; GUN/PAC armament switch - SAFE; LASER armament switch - SAFE; TGP - OFF; ALT SCE - As desired; HUD mode switch - As desired; CICU - OFF; JTRS - OFF; IFFCC switch - OFF
- [ ] **MFCD (L/R)** - OFF
- [ ] **Standby attitude indicator** - CAGE
- [ ] **Accelerometer** - Reset
- [ ] **Fire handles** - In
- [ ] **Fire extinguisher discharge switch** - Center position
- [ ] **Standby compass** - Check
- [ ] **Auxiliary landing gear extension handle** - In
- [ ] **Circuit breakers** - Check closed

## Engine Start

- [ ] **Left engine** - Start
- [ ] **Engine operate switch** - NORMAL (if used for motoring)
- [ ] **Left engine instruments & Hyd Pressure** - Check
- [ ] **Flight controls** - Check
- [ ] **Right engine** - Start. Repeat steps for right engine
- [ ] **Right engine instruments & Hyd Pressure** - Check
- [ ] **AC generators** - Check
- [ ] **Instrument inverter** - Check
