# A-10C Warthog Landing Checklist

## Before Landing

- [ ] **ATIS/Tower** - Contact and receive landing clearance
- [ ] **Fuel quantity** - Check sufficient for landing and reserve
- [ ] **Seat** - Armed
- [ ] **Harness** - Locked
- [ ] **Altimeter** - Set
- [ ] **Landing data** - Review (runway heading, length, elevation)

## Approach

- [ ] **Speed** - Reduce to approach speed (below 250 KIAS)
- [ ] **Landing gear** - DOWN (below 250 KIAS)
- [ ] **Landing gear lights** - Check three green
- [ ] **No gear warning horn** - Verify silent with gear down
- [ ] **Flaps** - Set as required (MVR or DN for full flaps)
- [ ] **Landing/taxi lights** - ON
- [ ] **Pitot heat** - ON
- [ ] **Speed brake** - As required
- [ ] **Approach speed** - Maintain (typically 165-185 KIAS depending on weight and configuration)
- [ ] **Final approach speed** - 140-165 KIAS (adjust for weight and conditions)
- [ ] **Angle of attack indexer** - Monitor for proper approach angle

## Landing

- [ ] **Runway alignment** - Verify on centerline
- [ ] **Touchdown aim point** - Visual acquisition
- [ ] **Power** - Adjust as required to maintain glideslope
- [ ] **Speed** - Maintain approach speed
- [ ] **Flare** - Initiate at appropriate height
- [ ] **Throttles** - IDLE after touchdown
- [ ] **Aerodynamic braking** - Maintain nose up attitude initially
- [ ] **Nose wheel** - Lower gently to runway
- [ ] **Wheel brakes** - Apply as required
- [ ] **Speed brake** - Extend (if not already extended)
- [ ] **Reverse thrust** - Not available (A-10 has no thrust reversers)
- [ ] **Directional control** - Maintain with rudder pedals/nose wheel steering
- [ ] **Exit runway** - At appropriate taxiway
- [ ] **Clear of active runway** - Confirm

## After Landing

- [ ] **Speed brake** - Retract
- [ ] **Flaps** - UP
- [ ] **Trim** - Reset to neutral
- [ ] **Landing/taxi lights** - As required
- [ ] **Strobe lights** - OFF
- [ ] **Anti-collision lights** - As required
- [ ] **Transponder** - As required
- [ ] **Pitot heat** - OFF (as required)
- [ ] **After landing checklist** - Complete during taxi
