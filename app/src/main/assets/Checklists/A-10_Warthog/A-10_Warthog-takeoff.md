# A-10C Warthog Takeoff Checklist

## Before Takeoff

- [ ] **Flight controls** - Check free and correct
- [ ] **Trim** - Set for takeoff
- [ ] **Flaps** - Set to takeoff position (MVR or UP as required)
- [ ] **Speed brake** - Retracted
- [ ] **Pitch trim** - 1-2 units nose up
- [ ] **Roll trim** - Centered
- [ ] **Yaw trim** - Centered
- [ ] **Engine instruments** - Check in green
- [ ] **Hydraulic pressure** - Check (2,800-3,350 psi)
- [ ] **Fuel quantity** - Check sufficient
- [ ] **Oxygen** - Check flow
- [ ] **Canopy** - Closed and locked
- [ ] **Seat** - Armed
- [ ] **Harness** - Locked
- [ ] **Flight instruments** - Set and checked
- [ ] **Altimeter** - Set
- [ ] **HSI** - Set
- [ ] **Heading** - Set
- [ ] **TGP** - Available for operation once STBY page is displayed (RIGHT MFCD OSB-11 Press)
- [ ] **STAT page** - Check status of aircraft systems, configure slew rates (RIGHT MFCD OSB-14 Press)
- [ ] **DSMS page** - Check all data from DTS was transferred correctly (no red indicators), manually adjust
- [ ] **Master ARM switch** - As required (SAFE for training)
- [ ] **IFF** - As required
- [ ] **Transponder** - ALT
- [ ] **Exterior lights** - As required
- [ ] **Landing/taxi lights** - ON
- [ ] **Strobe lights** - ON
- [ ] **Anti-collision lights** - ON

## Takeoff

- [ ] **Takeoff clearance** - Received
- [ ] **Throttles** - Advance smoothly to MAX
- [ ] **Engine instruments** - Monitor during takeoff roll
- [ ] **Airspeed** - Check alive
- [ ] **Abort decision speed** - Monitor
- [ ] **Rotation speed (130-145 KIAS)** - Rotate smoothly at appropriate speed (depends on weight)
- [ ] **Initial climb speed** - 200 KIAS minimum
- [ ] **Landing gear** - UP (after positive rate of climb)
- [ ] **Landing gear lights** - Check three green off
- [ ] **Flaps** - UP (after safe altitude and speed)
- [ ] **After takeoff checklist** - Complete when time permits
