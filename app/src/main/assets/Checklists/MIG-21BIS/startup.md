# MiG-21bis Startup Checklist

## 1. Cockpit Preparation

- [ ] **Canopy:** Close and seal the canopy.
  - *Note:* The canopy seal warning light above the seal lever should extinguish.
- [ ] **Battery:** Turn ON the main battery switch and the battery heat switch.
- [ ] **Inverters:** Turn ON the inverters.
- [ ] **Generators:** Turn ON the AC and DC generators.
- [ ] **Fuel Pumps:** Activate all three fuel pumps.
- [ ] **Fire Extinguisher:** Turn ON the fire extinguisher system.
- [ ] **APU:** Engage the Engine APU (Auxiliary Power Unit).
- [ ] **Throttle:** Unlock the throttle by clicking the engine stop lock lever.

## 2. Engine Start

- [ ] **Engine Start Button:** Press and hold the engine start button for approximately 4 seconds.
- [ ] **Monitor Engine:** Verify that the engine RPM increases and the "ENGINE START" indication appears on the master caution panel.
  - *Note:* Wait for engine RPM to start increasing. First stage compressor RPM should reach ~50% and second stage compressor RPM ~35%.
- [ ] **Engine Start Light:** Confirm the "ENGINE START" light extinguishes once the engine is running.

## 3. Post-Start & Systems Activation

- [ ] **Gyros:** Turn ON the two gyros.
- [ ] **Panel Switches:** Activate all switches on the upper right panel, except for the camera switch.
- [ ] **FDS Button:** Press and hold the FDS (Flight Direction System) button for 5 seconds to provide course correction.
- [ ] **IFF & RWR:** Turn ON the IFF (Identification Friend or Foe) and RWR (Radar Warning Receiver).
- [ ] **Radar:** Set to Standby.
- [ ] **ASP Sight:** Set the ASP sight to missile mode.
- [ ] **IR Rotary Switch:** Set the air IR rotary switch to position one and press button one.
- [ ] **Pitot Heat:** Turn ON the pitot heat.
- [ ] **Navigation Lights:** Set as required.
- [ ] **Landing/Taxi Light:** Set to Landing (up).
- [ ] **SAU Autopilot Control Panel:** Verify no lights are lit.

## 4. Final Checks

- [ ] **Gear Lever Fixator:** Unlock the gear lever fixator.
- [ ] **Guarded Switches:** Uncover the countermeasure guarded switch and the weapon guarded switch.
- [ ] **Flaps:** Set takeoff flaps.
- [ ] **Nosewheel Brake:** Set the nosewheel brake lever to ON (horizontal).
- [ ] **Main Air Pressure:** Check that the main air pressure gauge is above 10kg/cm².
