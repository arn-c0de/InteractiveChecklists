# AH-64D Apache Departure Checklist

## Pre-Takeoff Checks (Before Taxi)

- [ ] **Startup complete** - All systems operational
- [ ] **EUFD pages** - Review all systems green
  - Engine page: Both engines normal parameters
  - HYD page: Primary and Utility 2,850-3,250 psi
  - FUEL page: Sufficient fuel for mission
  - No caution/warning lights
- [ ] **MPD pages (Both stations)** - Verify configured
  - TSD: GPS lock, waypoints entered, route planned
  - WPN: Weapons inventory correct
  - FLT: Instruments operational, altimeter set
- [ ] **Flight controls (Pilot)** - Final check
  - Collective: Full range, friction adjusted
  - Cyclic: Full range all directions
  - Pedals: Full range both directions
  - Force trim: Release and test, verify holds position
- [ ] **Rotor RPM (EUFD)** - Verify 101% NP both engines
- [ ] **Canopy/doors** - Closed and latched
- [ ] **Seat harness (both crew)** - Locked tight
  - Lap belt tight
  - Shoulder harness locked
  - Quick release check
- [ ] **Parking brake** - Release when ready to taxi

## Taxi

### Before Taxi

- [ ] **Taxi clearance** - Obtain from tower/ground
  - "Ground, Apache 123, parking spot Alpha, ready to taxi, runway 27"
  - Receive: "Apache 123, taxi to runway 27 via taxiway Bravo, wind 270 at 10"
- [ ] **Taxi lights** - ON
  - Landing/taxi lights ON
  - Anti-collision lights ON
- [ ] **Taxi route** - Brief and verify
  - Review taxiway route to runway
  - Note obstacles, other traffic

### Taxi Technique (Pilot)

- [ ] **Hover taxi** - Normal method (0-10 ft AGL)
  - Collective: Raise smoothly to light on wheels
  - Continue raising collective until skids/wheels lift
  - Hover height: 3-5 ft AGL typical
  - Forward cyclic: Gentle inputs for forward movement (5-10 KIAS)
- [ ] **Rotor RPM** - Maintain 101% NP during taxi
  - Monitor continuously
  - If NP drops: Increase collective/power
- [ ] **Taxi speed** - Slow and controlled
  - Hover taxi: 5-10 KIAS (walking pace)
  - Surface taxi (if used): 10-20 KIAS maximum
- [ ] **Turns** - Coordinate pedals and cyclic
  - Pedals: Yaw nose in desired direction
  - Cyclic: Slight lateral toward turn to prevent drift
  - Smooth, gentle inputs
- [ ] **Clearance** - Maintain safe distance
  - Rotor disc clearance from obstacles: Minimum 25 ft
  - Other aircraft: Minimum 100 ft separation
  - Buildings, vehicles, personnel: Stay clear
- [ ] **Braking** - Use cyclic to slow/stop
  - Aft cyclic: Decelerates aircraft
  - Reduce collective: Settles to ground if stopping
- [ ] **Surface taxi (alternative)** - Wheels on ground
  - Used for long taxi or to conserve fuel
  - Collective: Lighten on wheels but don't lift off
  - Forward cyclic: Taxi on wheels at 10-20 KIAS
  - Less fuel burn than hover taxi

### Taxi Instruments (Monitor During Taxi)

- [ ] **EUFD** - Monitor
  - Engine parameters stable
  - NP: 101%
  - Transmission temperature/pressure normal
- [ ] **Heading** - Check matches taxiway direction
- [ ] **TSD** - Monitor position on airport diagram

## Run-Up (At Hold-Short Position)

### Position

- [ ] **Hold short of runway** - Position aircraft
  - Stop clear of runway hold-short line
  - Into wind if possible
  - Collective down, on ground or low hover
  - Set parking brake or hold position

### Before Takeoff Checklist

- [ ] **Flight controls** - Free and correct
  - Full cyclic deflection all directions
  - Full collective range
  - Full pedal range
  - No binding or restrictions
- [ ] **Force trim** - Test
  - Release trim (paddle switch on cyclic)
  - Move cyclic, release paddle
  - Verify new position holds
- [ ] **SAS (Stability Augmentation System)** - Verify engaged
  - Check SAS 1 and SAS 2 on FMC page (MPD)
  - No SAS caution lights
- [ ] **Engine instruments (EUFD)** - Check
  - NG: ~72% both engines
  - NP: 101% synchronized
  - TGT: Green band (650-750°C)
  - Oil pressure: 40-90 psi both engines
  - Oil temperature: Green band
- [ ] **Hydraulic systems (EUFD HYD page)** - Verify
  - Primary: 2,850-3,250 psi
  - Utility: 2,850-3,250 psi
  - Accumulators: Charged
- [ ] **Caution/Warning panel** - Clear (no lights)
- [ ] **Fuel quantity (EUFD FUEL page)** - Final check
  - Sufficient for flight + reserve
  - Fuel balance acceptable
- [ ] **Navigation (TSD)** - Verify
  - GPS lock (4+ satellites)
  - Present position correct
  - First waypoint active and correct
  - Route displayed
- [ ] **Radios** - Set and checked
  - Tower frequency active
  - Altimeter setting verified
  - Transponder: ALT (altitude reporting)
  - IFF: As required
- [ ] **Flight instruments (MPD FLT page)** - Check
  - Attitude indicator operational
  - Altimeter set correctly (QNH)
  - Airspeed zero
  - VSI zero
  - Heading correct
- [ ] **Lights** - Set for departure
  - Landing lights: ON
  - Anti-collision: ON
  - Position/nav lights: As required (ON at night)
- [ ] **Master arm (if applicable)** - SAFE
  - Weapons safed for peacetime departure
  - ARM only if tactical departure into threat area

### Takeoff Clearance

- [ ] **Request takeoff clearance** - Contact tower
  - "Tower, Apache 123, ready for takeoff, runway 27"
  - Receive: "Apache 123, cleared for takeoff runway 27, winds 270 at 10 gusting 15"
- [ ] **Read back clearance** - Verify
  - "Cleared for takeoff runway 27, Apache 123"
- [ ] **Final scan** - Both crew
  - Approach path clear
  - Runway clear
  - No traffic on final
  - Conditions acceptable

## Takeoff (Normal - No Obstacles)

### Hover and Position

- [ ] **Taxi onto runway** - Enter runway
  - Hover taxi to centerline
  - Align nose with runway heading
- [ ] **Position** - Center of runway
  - On centerline
  - Near approach end (to maximize runway available)
  - Into wind heading
- [ ] **Stabilize hover** - 3-5 ft AGL
  - Collective: Adjust for stable hover
  - Cyclic: Level attitude
  - Pedals: Aligned with centerline
  - Rotor RPM: 101% NP

### Takeoff Execution

- [ ] **Power check** - Verify sufficient power
  - At 3-5 ft hover, check power required
  - Torque: Should be well below 100% (typical 30-50% for hover)
  - Adequate power margin for climb
- [ ] **Cyclic forward** - Begin takeoff
  - Smooth forward cyclic input
  - Nose lowers slightly (5-10 degrees nose down)
  - Aircraft begins forward movement
- [ ] **Acceleration** - Gain airspeed on runway
  - Continue forward cyclic pressure
  - Maintain 3-5 ft AGL (skids/wheels just off ground)
  - Pedals: Maintain runway heading
  - Airspeed increasing: 0...20...40...60 KIAS
- [ ] **Translational lift** - 16-24 KIAS
  - Aircraft becomes more efficient
  - Noticeable increase in performance
  - Less power required to maintain altitude
  - May need to reduce collective slightly to maintain altitude
- [ ] **Effective translational lift (ETL)** - ~24 KIAS
  - Maximum efficiency gain
  - Reduced vibration
  - Power required decreases significantly
- [ ] **Climb initiation** - At 40-60 KIAS
  - Collective: Increase smoothly to climb power
  - Target climb speed: 60-80 KIAS
  - Cyclic: Aft slightly to establish climb attitude
  - Climb rate: 500-1,000 fpm typical

### Climb

- [ ] **Climb airspeed** - Establish and maintain
  - **Vy (Best Rate of Climb): 60 KIAS** - Fastest altitude gain
  - **Cruise climb: 80-100 KIAS** - Better forward progress, less steep
- [ ] **Climb power** - Monitor EUFD
  - Torque: Typically 60-80% (depends on weight)
  - TGT: Monitor, should stay in green band
  - NP: Maintain 101%
- [ ] **Climb attitude** - Maintain
  - Slight nose-up attitude (5-10 degrees)
  - Keep runway ahead in sight
- [ ] **Departure path** - Follow as cleared/briefed
  - Straight out, or
  - Climbing turn to crosswind/downwind as directed

## Takeoff (Tactical - High Performance)

### When to Use

- [ ] **Tactical departure** - Short field or obstacle clearance
  - Confined area
  - High obstacles (trees, wires, buildings)
  - Threat area (minimize time in vulnerable position)

### Execution

- [ ] **Hover** - Establish 3-5 ft AGL
  - On takeoff spot
  - Into wind if possible
  - Collective: Hover power set
- [ ] **Power check** - Verify maximum power available
  - Collective: Increase to check power margin
  - Verify sufficient power for vertical climb
  - Reduce collective back to hover
- [ ] **Collective pull** - Maximum power
  - Smoothly increase collective to maximum allowable torque
  - Torque: Up to 100% (or aircraft limit)
  - TGT: Monitor, do NOT exceed red line (810°C)
- [ ] **Vertical climb** - Straight up
  - Cyclic: Neutral (may need slight forward to prevent rearward drift)
  - Pedals: Maintain heading
  - Climb rate: 1,000-2,000+ fpm (depends on weight and power)
- [ ] **Obstacle clearance** - Gain altitude
  - Climb vertically until clear of obstacles
  - Typical: 50-100 ft above obstacles
- [ ] **Transition to forward flight** - After obstacle clearance
  - Cyclic forward to gain airspeed
  - Transition through ETL (24 KIAS)
  - Accelerate to climb speed (60-80 KIAS)
  - Collective: Adjust to maintain climb or level off as required

## Takeoff (Running Takeoff - Heavy Weight)

### When to Use

- [ ] **Heavy gross weight** - Insufficient power to hover
  - Full fuel and heavy weapons load
  - High density altitude (hot day, high elevation)
  - Insufficient power for vertical or hover takeoff

### Execution

- [ ] **Position on runway** - Far back to maximize runway
  - Use full runway length available
  - Aligned with centerline
- [ ] **Skids/wheels on ground** - Start from ground
  - Collective: Down, aircraft on ground
- [ ] **Collective increase** - Smoothly apply power
  - Increase collective to lighten on wheels but don't lift
  - Reduce weight on landing gear
- [ ] **Cyclic forward** - Begin ground roll
  - Forward cyclic for acceleration
  - Aircraft rolls forward on wheels
  - Similar to airplane takeoff
- [ ] **Acceleration** - Gain speed on ground
  - Airspeed increasing: 0...10...20...30 KIAS
  - Pedals: Maintain runway heading
  - Aircraft still on wheels/skids
- [ ] **Lift-off** - At ETL speed (~24 KIAS)
  - Continue increasing collective
  - Aircraft becomes light on wheels
  - Lifts off naturally as speed increases
  - Transition to airborne
- [ ] **Climb** - Accelerate and climb
  - Accelerate to climb speed (60-80 KIAS)
  - Establish climb attitude
  - Climb rate: May be reduced due to heavy weight

## After Takeoff

### Climb Out (Below 500 ft AGL)

- [ ] **Airspeed** - Maintain climb speed
  - 60-80 KIAS typical
- [ ] **Rotor RPM (NP)** - Monitor 101%
- [ ] **Engine instruments (EUFD)** - Monitor
  - TGT: Watch for over-temp (especially if high power)
  - Torque: Adjust collective if needed
  - Oil pressure/temp: Green band
- [ ] **Flight path** - Maintain departure course
  - Runway heading, or as cleared by ATC
  - Clear of obstacles
- [ ] **Altitude** - Climb to pattern altitude or as cleared
  - Pattern altitude: Typically 500 ft AGL
  - Or as assigned by ATC
  - Or cruise altitude for cross-country

### Level-Off (At Assigned/Desired Altitude)

- [ ] **Collective** - Reduce to level flight power
  - As approaching altitude, reduce climb rate
  - Level off smoothly
  - Adjust collective to maintain altitude
- [ ] **Airspeed** - Accelerate to cruise
  - Cyclic forward to accelerate
  - Cruise speed: 100-140 KIAS (mission dependent)
  - Fast cruise: 140-160 KIAS (reduced endurance)
- [ ] **Trim** - Adjust for hands-off flight
  - Use force trim (paddle switch) to reduce control forces
  - Set trim for cruise flight

### After Takeoff Checklist

- [ ] **Landing lights** - OFF (once clear of airport area)
  - Unless required for traffic/visibility
- [ ] **Departure frequency** - Contact as required
  - Switch from tower to departure control
  - Or switch to tactical frequency if military mission
- [ ] **Transponder** - Verify ALT mode
- [ ] **Navigation** - Activate route
  - TSD: Verify tracking to first waypoint
  - Autopilot (if used): Engage as appropriate
- [ ] **Fuel** - Start monitoring consumption
  - Note fuel quantity at takeoff
  - Monitor fuel flow and endurance
- [ ] **Mission systems** - Configure as required
  - If tactical mission: Master arm, weapons hot, ASE active
  - If training: Systems as briefed

## Departure Patterns

### Straight-Out Departure

- [ ] **Climb straight ahead** - Maintain runway heading
  - Continue on runway heading until instructed otherwise
  - Typical for IFR or when traffic requires
- [ ] **Altitude** - Climb to assigned
  - ATC may assign: "Climb and maintain 3,000 feet"
  - Or: "Maintain VFR at or below 1,500 feet"

### Pattern Departure (Remaining in Traffic Pattern)

- [ ] **Crosswind turn** - At departure end or as briefed
  - Climbing turn 90 degrees from runway heading
  - Altitude: 500 ft AGL typically
- [ ] **Downwind leg** - Parallel to runway
  - Altitude: 500 ft AGL
  - Airspeed: 60-80 KIAS
  - Spacing: ~1/2 mile from runway

### Tactical Departure (Low Level)

- [ ] **Climb to tactical altitude** - Stay low
  - 50-200 ft AGL depending on terrain/threats
  - Minimize radar/visual detection
- [ ] **Accelerate to cruise** - Low level speed
  - 100-140 KIAS
- [ ] **Terrain masking** - Use terrain for concealment
  - Follow valleys, behind ridges
  - Avoid skylining
- [ ] **Navigation** - TSD monitoring
  - Low-level navigation to first waypoint
  - Continuous terrain awareness

## Rejected Takeoff (Abort on Ground)

### When to Abort

- [ ] **Abort decision** - Before becoming airborne
  - Engine failure/malfunction
  - Caution/warning light
  - Control malfunction
  - Obstruction on runway
  - Any unsafe condition

### Abort Procedure

- [ ] **Collective** - Lower immediately
  - Reduce power, settle to ground
- [ ] **Cyclic** - Aft to stop forward movement
  - Decelerate rapidly
  - Come to stop on runway
- [ ] **Pedals** - Maintain runway heading
  - Prevent yaw
- [ ] **Power** - Reduce to flight idle
  - Both engines to idle
- [ ] **Position** - Stop clear of runway if able
  - Taxi clear if aircraft controllable
  - Or remain on runway and request assistance
- [ ] **Tower notification** - Advise ATC
  - "Tower, Apache 123, aborting takeoff, [state reason]"
- [ ] **Systems check** - Assess problem
  - Determine if can continue or require maintenance

## Emergency After Takeoff (Below 500 ft)

### Engine Failure on Takeoff

- [ ] **Immediate action** - LAND IMMEDIATELY
  - Below 500 ft AGL = insufficient altitude for safe autorotation
  - Do NOT attempt to return to runway
- [ ] **Collective** - Lower immediately (enter autorotation)
  - Maintain rotor RPM (101% NP)
- [ ] **Landing area** - Straight ahead
  - Land on remaining runway if available
  - Or land in clear area ahead
  - Do NOT turn back (altitude too low)
- [ ] **Mayday call** - If time permits
  - "MAYDAY, Apache 123, engine failure after takeoff"
- [ ] **Flare and land** - Execute autorotation landing
  - Flare to reduce speed
  - Collective application at touchdown
  - Aim for survivable landing

### Malfunction After Takeoff (Non-Critical)

- [ ] **Assess severity** - Can flight continue?
  - Minor issue: Continue to pattern, land as soon as practical
  - Major issue: Land immediately ahead
- [ ] **Tower notification** - Report problem
  - "Tower, Apache 123, experiencing [problem], request immediate landing"
- [ ] **Landing** - Return to field or land ahead
  - If altitude/position permits: Return to runway
  - If low altitude: Land ahead in clear area
  - Follow emergency checklist for specific malfunction
