# AH-64D Apache Air-to-Ground Combat Checklist

## Pre-Combat Preparation

- [ ] **Mission brief** - Review with crew
  - Target description and coordinates
  - ROE (Rules of Engagement)
  - Threat assessment (AAA, MANPADS, SAMs)
  - Friendly positions (deconfliction)
  - Egress routes (primary and alternate)
- [ ] **Crew coordination** - Confirm roles
  - CPG: Target acquisition, weapons employment, threat monitoring
  - Pilot: Aircraft control, navigation, situational awareness
- [ ] **Weapons load verification (WPN page)** - Check
  - 30mm gun rounds (typically 1,200 max)
  - Hellfire missiles (quantity and type)
  - Rockets (quantity and type)
- [ ] **Fuel state** - Verify sufficient
  - Combat + egress + 20 min reserve minimum
- [ ] **Battle positions (BP)** - Identify on TSD
  - Primary and alternate firing positions
  - Terrain masked, good fields of fire
  - Covered egress routes
- [ ] **FARP location** - Note for emergency rearm/refuel

## Combat Systems Activation (CPG Station)

- [ ] **Master arm switch (Right console)** - ARM (guard up, switch forward)
  - MASTER ARM caution light extinguishes when armed
- [ ] **WPN page on MPD** - Select and verify
  - All weapons show READY status
  - No red X's or faults
- [ ] **FCR (Fire Control Radar)** - Set mode as required
  - **GTM (Ground Targeting Mode)** - For Hellfire/target acquisition
  - **TPM (Terrain Profile Mode)** - For terrain mapping
  - **RMAP (Real-beam MAP)** - For wide area search
  - Switch modes using FCR panel or WPN page buttons
- [ ] **TADS (Target Acquisition Designation Sight)** - Active
  - DTV (Day TV) or FLIR (Forward Looking Infrared) selected
  - Laser armed for ranging/designation
  - TADS slaved to FCR or manual control
- [ ] **ASE (Aircraft Survivability Equipment)** - All systems active
  - RWR (Radar Warning Receiver) - ON, volume set
  - CMWS (Missile Warning) - ON
  - LWS (Laser Warning) - ON
  - Chaff/Flare program loaded and ready
- [ ] **Laser code** - Set and verify
  - CPG and Pilot stations matched (e.g., 1688)
  - If working with other aircraft/JTACs, coordinate codes
- [ ] **Gun selected** - If primary weapon
  - Press GUN button on WPN page
  - Burst limiter set (10, 20, 50, or CONT)
  - Azimuth limits set if required

## Target Area Entry

- [ ] **Battle position (BP)** - Occupy
  - Use terrain masking (remain behind ridge/treeline)
  - Altitude: As low as tactically safe (minimize exposure)
  - Minimize time in exposure (quick peek, shoot, hide)
- [ ] **Aircraft positioning (Pilot)** - Establish
  - Hover or slow forward flight (0-40 KIAS) for precision
  - Altitude: Depends on terrain and standoff (typically 50-200 ft AGL)
  - Heading: Nose toward target area
- [ ] **ASE monitoring** - Continuous
  - Both crew monitor for RWR/LWS/CMWS alerts
  - Call out threats immediately
- [ ] **Communication** - Maintain
  - Wingman coordination
  - Ground forces coordination (if CAS mission)
  - TOC/Command updates

## FCR Target Acquisition (CPG)

- [ ] **FCR mode** - Select GTM (Ground Targeting Mode)
  - Press GTM button on FCR panel or WPN page
- [ ] **FCR scan** - Initiate
  - Press FCR SCAN button or NTS (Next To Shoot) trigger on TEDAC
  - FCR scans 90-degree sector ahead of aircraft
  - Range: Up to 8 km for vehicles, 4 km for dismounts
- [ ] **Target detection** - FCR displays contacts on TSD
  - Moving targets prioritized (MTI - Moving Target Indicator)
  - Stationary targets also detected
- [ ] **Target sorting** - Evaluate contacts
  - Size, speed, location
  - Threat priority (tanks, APCs, AAA, etc.)
- [ ] **Target selection** - CPG chooses target
  - Use TEDAC cursor or NTS trigger to cycle through targets
  - Selected target designated with acquisition cursor
- [ ] **Target handoff to TADS** - Automatic
  - TADS automatically slaves to FCR-designated target
  - TADS image shows target in crosshairs
- [ ] **Visual target identification** - CPG confirms via TADS
  - DTV (Day TV) or FLIR imagery
  - Zoom in for positive ID (NARROW field of view)
  - Confirm target is hostile, not friendly

## TADS Manual Target Acquisition (If Not Using FCR)

- [ ] **TADS control** - CPG controls via TEDAC
  - TADS LOS (Line of Sight) controlled by hand grips or slew
- [ ] **TADS field of view** - Select appropriate
  - WIDE (30° FOV) - Initial search
  - MEDIUM (4-6° FOV) - Tracking
  - NARROW (1.2° FOV) - ID and engagement
  - ZOOM (0.4° FOV) - Fine detail
  - Press FOV button on TEDAC to cycle
- [ ] **Sensor select** - Day or night
  - DTV (Day TV) - Daytime operations
  - FLIR (Infrared) - Night or low visibility
  - Toggle on TEDAC panel
- [ ] **Target search** - Scan suspected area
  - Use TADS hand controller to slew LOS
  - Look for movement, thermal signatures (FLIR), shapes
- [ ] **Target identification** - Zoom in for positive ID
  - Switch to NARROW or ZOOM FOV
  - Confirm hostile target per ROE
- [ ] **Laser range** - Determine distance to target
  - Press laser trigger (first detent) on TEDAC
  - Range displays on TADS image and WPN page
  - Critical for accurate weapons employment

## Hellfire Missile Employment

### Hellfire Selection and Setup

- [ ] **Hellfire selected** - Press HF button on WPN page
  - Hellfire becomes active weapon
  - HF symbology appears on TADS and IHADSS
- [ ] **Hellfire type verification** - Check WPN page
  - **AGM-114K (SAL - Semi-Active Laser)** - Most common
    - Requires continuous laser designation until impact
  - **AGM-114L (RF - Radar Frequency/Millimeter Wave)** - Fire and forget
    - No laser required, autonomous seeker
- [ ] **Missile quantity** - Check WPN page
  - Typically 8-16 missiles loaded
  - Note which rails loaded (L1-L4, R1-R4)
- [ ] **Missile constraints check** - Verify WPN page
  - No red X's or faults on missile icons
  - Seeker status READY
  - If SAL Hellfire: Laser code matches (CPG and missile)

### SAL (Semi-Active Laser) Hellfire Employment

- [ ] **Range to target** - Laser range via TADS
  - Optimal range: 2-8 km (1.2-5 nm)
  - Minimum range: 500 m (ballistic constraint)
  - Maximum range: 8 km (typical), up to 11 km (extreme)
- [ ] **Target designation** - TADS locked on target
  - TADS crosshairs on target aimpoint
  - Press and hold laser trigger (first detent) on TEDAC
  - Laser fires and ranges target
- [ ] **Acquisition source** - Verify TADS tracking
  - ACQ SRC shows TADS on WPN page
  - Missile seeker slaved to TADS LOS
- [ ] **Launch constraints** - Check WPN page
  - No LOBL (Lock-On Before Launch) warnings
  - Clear field of fire (no obstacles between aircraft and target)
  - Clear missile flight path
- [ ] **Missile launch** - Execute
  - Continue holding laser trigger (keeps laser on target)
  - Press weapon trigger (second detent) on TEDAC
  - Missile launches from selected rail
  - "MISSILE AWAY" audio callout
  - CPG: Call "Missile away" to Pilot
- [ ] **Laser designation maintain** - CRITICAL FOR SAL
  - Keep laser trigger held (laser must stay on target)
  - Keep TADS crosshairs on target aimpoint
  - Do not break lock until missile impact (~10-20 seconds)
  - TADS will track laser spot, missile homes on spot
- [ ] **Missile flight monitoring** - Observe
  - Missile will appear as streak toward target
  - Time of flight: ~10-15 seconds at 5 km range
- [ ] **Impact** - Confirm
  - Visual or FLIR confirmation of impact
  - Release laser trigger after impact
- [ ] **BDA (Battle Damage Assessment)** - Assess
  - Target destroyed/damaged/intact?
  - Re-attack if necessary
  - Select next target or re-engage

### RF (Radar Frequency) Hellfire Employment (AGM-114L)

- [ ] **RF Hellfire selected** - Verify type on WPN page
- [ ] **Range to target** - Via TADS or FCR
  - Optimal: 3-8 km
  - Maximum: 8-11 km
- [ ] **Target acquisition** - FCR or TADS
  - RF Hellfire uses own millimeter wave seeker
  - Lock-on achieved via FCR handoff or TADS pointing
- [ ] **LOBL (Lock-On Before Launch)** - Verify
  - Seeker locks onto target before launch
  - LOBL indication on WPN page
- [ ] **Launch** - Fire and forget
  - Press weapon trigger on TEDAC
  - Missile away
  - No laser required - missile autonomous
  - Pilot can maneuver immediately after launch
- [ ] **BDA** - Assess results

### Ripple/Salvo Hellfire Employment

- [ ] **Multiple targets** - If engaging several targets quickly
- [ ] **Ripple mode** - Set on WPN page
  - Single: One missile per trigger pull
  - Ripple: Multiple missiles in sequence
  - Set ripple quantity (2, 4, etc.)
  - Set ripple interval (0.5-3 seconds between launches)
- [ ] **Engagement** - Execute
  - Designate first target with TADS/laser
  - Launch first missile
  - Maintain laser on first target until impact
  - After first impact, slew TADS to next target
  - Repeat for subsequent missiles

## 30mm M230 Chain Gun Employment

### Gun Selection and Setup

- [ ] **Gun selected** - Press GUN button on WPN page
  - Gun symbology appears on TADS and IHADSS (Pilot)
- [ ] **Gun status** - Check WPN page
  - Rounds remaining (max 1,200)
  - Gun system READY (no faults)
- [ ] **Gun sight** - TADS or Fixed Forward
  - **TADS slaved** - Gun points where TADS looks (±110° azimuth)
  - **Fixed forward** - Gun fixed, pilot aims aircraft
  - Toggle on WPN page or weapons panel
- [ ] **Burst limiter** - Set on WPN page
  - 10 rounds - Precision, conserve ammo
  - 20 rounds - Standard engagement
  - 50 rounds - Heavy targets
  - CONT (Continuous) - Unlimited burst (trigger held)

### Gun Employment - TADS Slaved

- [ ] **Range to target** - Laser range via TADS
  - Effective range: 800-1,500 m optimal
  - Maximum effective: 1,500-1,700 m
  - Minimum safe: 200 m (fragmentation risk)
- [ ] **Target designation** - TADS on target
  - TADS crosshairs on target aimpoint
  - Gun turret automatically slews to TADS LOS
- [ ] **Ballistic solution** - Automatic
  - Gun computer calculates lead, superelevation
  - TADS reticle shows aim point
- [ ] **Firing** - Execute
  - Press weapon trigger on TEDAC
  - Observe tracer rounds (every 5th round)
  - Adjust aim based on impact
- [ ] **Ammunition management** - Monitor rounds
  - Check remaining count on WPN page
  - Conserve ammo for multiple targets
  - Call "Winchester guns" when out

### Gun Employment - Fixed Forward (Pilot)

- [ ] **Gun mode** - Set to FIXED on WPN page
- [ ] **IHADSS gun reticle** - Pilot sees gun sight in helmet display
- [ ] **Aircraft pointing** - Pilot aims aircraft at target
  - Use cyclic to put reticle on target
  - Account for range and ballistics
- [ ] **Firing** - Pilot fires
  - Gun trigger on cyclic grip
  - Observe tracers, adjust aim
- [ ] **Crew coordination** - Both crew working
  - CPG: Ranging, target ID, next target search
  - Pilot: Aircraft control and gun employment

## 2.75" Rocket Employment

### Rocket Selection and Setup

- [ ] **Rockets selected** - Press RKT button on WPN page
  - Rocket symbology on TADS and IHADSS
- [ ] **Rocket inventory** - Check WPN page
  - Launcher type: M261 (19 rockets) or M260 (7 rockets)
  - Typical load: 38-76 rockets (2-4 launchers)
  - Rocket type displayed:
    - **M151 HE (High Explosive)** - Point targets, anti-personnel
    - **M229 HE** - Area effect
    - **M255 Flechette** - Anti-personnel, area saturation
    - **M156 WP (White Phosphorus)** - Smoke/marking
    - **M261 MPSM (Multi-Purpose Sub-Munition)** - Anti-armor/personnel
- [ ] **Rocket quantity selector** - Set salvo size
  - 1, 2, 5, 10, 20, ALL
  - Depends on target size and type
- [ ] **Rocket zone** - Set if multiple launcher types
  - LEFT, RIGHT, or BOTH

### Rocket Employment - TADS Designated

- [ ] **Range to target** - Laser range via TADS
  - Minimum safe: 500 m (fragmentation danger)
  - Optimal: 1,500-3,000 m
  - Maximum effective: 4,000-6,000 m (depends on rocket type)
- [ ] **Target acquisition** - TADS on target area
  - Rockets are area weapons (not precision)
  - Aim at center of target area
- [ ] **Ballistic solution** - Automatic
  - Rocket computer calculates trajectory
  - TADS or IHADSS reticle shows aim point
- [ ] **Launch constraints** - Verify
  - Clear of obstacles
  - Safe distance from friendlies (danger close: 600 m minimum)
- [ ] **Launch** - Execute
  - Press weapon trigger on TEDAC
  - Rockets fire in rapid sequence (ripple rate: ~6 rounds/sec)
  - Massive launch signature (smoke, flash)
- [ ] **Immediate maneuver** - After launch
  - Pilot: Break/mask immediately if threat present
  - Rocket launch very visible to enemy
- [ ] **BDA** - Assess impacts
  - Observe impact pattern (should be grouped)
  - Assess target area damage

### Rocket Employment - Fixed Forward (Pilot IHADSS)

- [ ] **Rocket mode** - IHADSS reticle displayed for pilot
- [ ] **Dive/run-in** - Establish attack profile
  - Airspeed: 80-120 KIAS
  - Dive angle: 10-20 degrees typical
  - Release altitude: 500-1,500 ft AGL (depends on range desired)
- [ ] **Aiming** - Pilot uses IHADSS reticle
  - Point reticle at target
  - Account for range and ballistics (reticle shows calculated impact)
- [ ] **Launch and break** - Execute attack
  - Fire rockets (trigger on cyclic)
  - Immediate break and egress after launch

## Defensive Maneuvers (Under Fire)

- [ ] **RWR warning** - Radar threat detected
  - Audio tone and visual on ASE page
  - Identify threat type and bearing
  - **Actions:**
    - Immediate maneuver toward threat (beam or defeat geometry)
    - Deploy chaff (press CHAFF button or auto-program)
    - FCR shutdown if radar-guided SAM threat
    - Mask behind terrain
- [ ] **LWS warning** - Laser designation detected
  - Audio warning: "LASER LASER LASER"
  - Laser from enemy designator or rangefinder
  - **Actions:**
    - Immediate break and mask
    - Deploy smoke/flares
    - Change altitude/aspect
    - Locate and engage threat if able
- [ ] **CMWS warning** - Missile launch detected
  - Audio warning: "MISSILE MISSILE MISSILE"
  - Visual on ASE page with bearing
  - **Actions:**
    - Immediate break toward threat (beam missile)
    - Deploy flares (auto or manual)
    - Nose low, max power, mask terrain
    - Countermaneuver (high-G turn)
- [ ] **Ground fire (AAA/Small arms)** - Tracers/impacts
  - **Actions:**
    - Immediate jink/break
    - Gain speed and mask
    - Change altitude unpredictably
    - Return fire if target identified
    - Egress if heavily engaged

## Battle Damage Assessment and Re-Attack

- [ ] **BDA after each engagement** - Assess
  - Target destroyed? (secondary explosion, fire, no movement)
  - Target damaged? (mobility kill, partial destruction)
  - Target intact/miss? (no visible effect)
- [ ] **Re-attack decision** - If target survives
  - Remaining weapons available?
  - Fuel state sufficient?
  - Threat level acceptable?
  - Time available?
- [ ] **Re-attack execution** - Engage again
  - Different weapon if first ineffective
  - Different attack angle/profile
  - Coordinate with wingman for combined attack

## Egress from Target Area

- [ ] **Mission complete/abort** - Decision to egress
  - Objectives met
  - Winchester (out of ammo)
  - Bingo fuel
  - Heavy damage
  - Threat too high
- [ ] **Master arm** - SAFE (guard down)
- [ ] **Egress route** - Execute planned route
  - Use terrain masking
  - Vary altitude and heading
  - Maximum speed compatible with terrain flight
- [ ] **Fuel check** - Verify sufficient for RTB
- [ ] **Damage assessment** - Check aircraft
  - Flight controls responsive?
  - Engine instruments normal?
  - Hydraulics normal?
  - Any unusual vibrations/noises?
- [ ] **Communication** - Report status
  - Wingman: Position, status, fuel
  - TOC/Command: Mission results, BDA, RTB
