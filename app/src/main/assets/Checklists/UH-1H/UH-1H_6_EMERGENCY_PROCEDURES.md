# UH-1H Emergency Procedures (Essentials)

> **Warning:** These are concise checklist prompts. Always follow the published AFM/NATOPS and unit emergency procedures for full steps and limitations.

## Example Emergency Radio Calls

- **Mayday (immediate danger)** - "Mayday, Mayday, Mayday. UH-1H [Callsign]. Engine failure. Position [lat/long or landmark], altitude [xx ft], number of souls on board [xx], request immediate assistance. Over."
- **Pan-Pan (urgent but not immediately life-threatening)** - "Pan-Pan, Pan-Pan, Pan-Pan. UH-1H [Callsign]. [Nature of problem], position [location], request assistance."
- **Radio failure / transponder** - "[Callsign] declaring radio failure, squawking 7600, proceeding to [plan]."

## Engine Failure (In Flight) - Autorotation

- [ ] **Control aircraft** - Establish autorotation, maintain rotor RPM in green band
- [ ] **Airspeed** - Establish recommended autorotation airspeed (refer AFM)
- [ ] **Collective** - Reduce to maintain rotor RPM
- [ ] **Transponder** - SQUAWK 7700 (XPNDR)
- [ ] **Mayday** - Transmit on appropriate radio (VHF/UHF)
- [ ] **Prepare for landing** - Select suitable landing area, brief crew
- [ ] **Flare / Cushion** - At landing, increase collective to cushion (per AFM)

## Engine Fire In Flight

- [ ] **Throttle / Power** - IDLE / OFF (as required by procedures)
- [ ] **Fuel Valve** - OFF (FUEL VALVE)
- [ ] **Fire Extinguisher** - ACTIVATE if equipped and trained
- [ ] **Emergency Shutdown** - Perform if required (condition levers / fuel off)
- [ ] **Land as soon as practicable** - Select landing site and expedite

## Electrical Fire / Smoke

- [ ] **Master / Battery switches** - OFF as required (BAT / GEN switches)
- [ ] **Vents / Windows** - Open as needed for ventilation
- [ ] **Fire Extinguisher** - DISCHARGE (if source located and safe)
- [ ] **Land as soon as practicable** - Prepare crew and passengers

## Transmission / Drive System Warning

- [ ] **Collective** - Adjust to minimize load
- [ ] **Land Immediately** - If indications of severe transmission problem
- [ ] **Emergency Shutdown on ground** - After landing, follow shutdown procedure

## Ditching / Forced Water Landing

- [ ] **Crew brief** - Prepare for ditching (brace positions, exits)
- [ ] **Doors / Windows** - Secure per procedure
- [ ] **Transponder** - SQUAWK 7700
- [ ] **Life rafts / Survival gear** - Prepare

## Emergency Shutdown (On Ground)

- [ ] **Condition levers / Fuel** - OFF
- [ ] **Battery / Generator switches** - OFF
- [ ] **Ignition keylock** - Remove key as required
- [ ] **Fire extinguisher** - READY / DISCHARGE if needed

> **Note:** These procedures are condensed prompts to guide the crew; always consult and follow the UH-1H AFM/NATOPS and local unit SOP for exact sequence, limits and safety cautions.
