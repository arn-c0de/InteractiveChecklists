# UH-1H Landing Checklist

## Approach Briefing

- [ ] **Crew briefing** - Complete (landing zone, winds, obstacles, technique)
- [ ] **Landing type** - Normal / Confined / Tactical (brief specifics)

## Before Landing (Approach)

- [ ] **Landing Light** - ON (Landing light switch)
- [ ] **Position/Anti-Collision Lights** - Set as required (EXT LTS)
- [ ] **Fuel** - Verify quantity and fuel valve position (FUEL VALVE - ON)
- [ ] **Transponder** - Set code / MODE to ALT (XPNDR)
- [ ] **NAV/COM** - Set inbound frequency (NAV/COM 1/2)
- [ ] **Flight Instruments** - Check (Attitude, VSI, Altimeter set, Airspeed)
- [ ] **Rotor & Engine Instruments** - Check N1/Nf, torque, oil temp / press
- [ ] **Collective** - Check friction and range
- [ ] **Doors/Canopy** - Secure / locked
- [ ] **Seat harness** - Locked and tight

## Final Approach / Flare

- [ ] **Airspeed** - Maintain approach speed (reference AFM or unit SOP)
- [ ] **Collective** - Smoothly adjust to control descent
- [ ] **Pedals / Heading** - Maintain runway/aim point alignment
- [ ] **Visual references** - Confirm aiming point and obstacles

## Touchdown

- [ ] **Collective** - Increase smoothly to cushion touchdown
- [ ] **Brakes / Parking Brake** - SET as required
- [ ] **Rotors** - Monitor Np/Nf during settling

## After Landing

- [ ] **Transponder** - STBY or as required
- [ ] **Position / Anti-Collision Lights** - OFF as required
- [ ] **Avionics / Radios** - OFF or set per local SOP
- [ ] **Secure aircraft** - Chocks / tie downs as required

## Example Radio Calls

- **Approach inbound (ATC/Tower)** - "Tower, UH-1H [Callsign], inbound for landing, LZ/runway [ID], full stop."
- **LZ call (tactical / ground)** - "LZ Control, UH-1H [Callsign], inbound 1 mile, north, landing hot/rolling in."
- **Clear of LZ/runway** - "Tower/LZ, UH-1H [Callsign], clear of landing area, taxiing to parking."

> **Note:** Use unit SOP and AFM/NATOPS references for precise speeds, power settings and limitations.
