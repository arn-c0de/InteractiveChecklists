# UH-1H Engine Shutdown Checklist

> NOTE: Steps 3 through 8 are to be completed after the last flight of the day if system operation was not verified during the mission.

## Shutdown Procedure

- [ ] **Throttle** - Idle for two minutes
- [ ] **FORCE TRIM** - Check ON
- [ ] **PITOT HTR** - Check: place switch ON; note loadmeter increase; then OFF
- [ ] **INVTR switch** - OFF
  - Check INST INVERTER caution light illumination; switch to SPARE, check caution light OFF
- [ ] **AC voltmeter** - Check 112–118 V
- [ ] **MAIN GEN** - OFF
  - DC GENERATOR caution light should illuminate; standby generator loadmeter should indicate a load

## Non-Essential Bus Checks (if installed)

- [ ] **VM switch** - NON-ESS BUS
- [ ] **NON-ESS-BUS switch** - Manual ON; check DC volts same as Step 6
- [ ] **VM SWITCH** - ESS BUS

## Final Power & Panels

- [ ] **MAIN GEN** - ON and guard closed (DC GENERATOR caution light out; main generator loadmeter indicates a load)
- [ ] **STARTER GEN** - START
- [ ] **Throttle** - OFF
- [ ] **Center Pedestal switches**
  - [ ] **FUEL** - OFF
  - [ ] **Avionics** - OFF
- [ ] **Overhead switches**
  - [ ] **INVTR** - OFF
  - [ ] **PITOT HTR** - OFF
  - [ ] **EXT LTS** - OFF
  - [ ] **CABIN HEATING** - OFF
  - [ ] **Indoor Lighting** - OFF
  - [ ] **BAT** - OFF
- [ ] **Ignition keylock** - Remove key as required

> **Note:** Complete post-flight checks and secure aircraft as required by unit SOP.
