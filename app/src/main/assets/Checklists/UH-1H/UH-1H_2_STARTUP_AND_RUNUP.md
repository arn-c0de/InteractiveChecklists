# UH-1H Startup & Runup Checklist

## Startup - Preparations

- [ ] **Fireguard** - Posted if available
- [ ] **Rotor blades** - Check clear and untied
- [ ] **Ignition key lock** - ON
- [ ] **Throttle** - Set for start (near decrease side to engine idle stop)

## Engine Start

- [ ] **Start switch** - Press and hold; note start time and DC voltmeter
  - Maximum starter energize time: **40 seconds**
  - Cooling time between starts: **3 minutes**
  - Maximum attempts: **3 in one hour**
- [ ] **Main rotor** - Verify rotor turning as N1 reaches 15% (abort if NOT turning)
- [ ] **START FUEL** - OFF at 40% N1
- [ ] **Release start switch** - At 40% N1 or after 40 seconds
- [ ] **Advance throttle** - Slowly past idle stop to engine idle; check idle stop
- [ ] **Gas Producer (N1)** - Check 68–72% (hold slight pressure against idle stop)

## Post-Start Checks

- [ ] **INVTR switch** - ON
- [ ] **Engine and transmission oil pressures** - Check
  - **Engine oil pressure:** Idle ≥25 PSI; Continuous 80–100 PSI; Max 100 PSI
  - **Transmission oil pressure:** Min 30 PSI; Continuous 40–60 PSI; Max 70 PSI

## Engine Runup

- [ ] **Avionics** - ON
- [ ] **STARTER GEN** - STBY GEN
- [ ] **Systems check** - Fuel, engine, transmission, electrical
  - AC: 112–118 V
  - DC: ~27–28.5 V depending on temperature
- [ ] **Throttle** - Set to full; low RPM audio/light should be off at 6100–6300 RPM
  - Check for **6600 RPM**
- [ ] **Avionics & flight instruments** - Check and set as required

## Hover Check

- [ ] **Engine & transmission instruments** - Check
- [ ] **Flight instruments** - Check (VSI, altimeter, slip indicator, compass, attitude, airspeed)
- [ ] **Power check** - Compare indicated torque required to hover with performance charts

## Example Radio Calls

- **Ready to taxi / ready for takeoff** - "Ground/Tower, UH-1H [Callsign], engines running, ready to taxi/to takeoff."
- **Rolling / Takeoff call** - "Tower, UH-1H [Callsign], rolling for departure runway [#], [intentions]."

> **Note:** Ensure all engine parameters are stable and within limits before takeoff.
