# UH-1H Navigation Checklist

## Preflight Navigation Setup

- [ ] **Flight plan** - Briefed and loaded (if using GPS/DTS)
- [ ] **Altimeter** - Set to local QNH (BARO knob)
- [ ] **Heading bug** - Set to planned heading
- [ ] **HDG / CRS selectors** - Verify positions

## Radios & Avionics

- [ ] **NAV/COM 1 / 2** - Set frequencies (NAV/COM knobs)
- [ ] **ADF** - Set frequency / verify bearing (ADF selector)
- [ ] **GPS** - Power ON, verify position (GPS Power / NAV unit)
- [ ] **Transponder (XPNDR)** - Set code, MODE (ALT)
- [ ] **Intercom (ICS)** - Confirm crew communication

## In-Flight Navigation Checks

- [ ] **VOR / OBS** - Tune and verify radial (VOR OBS knob)
- [ ] **GPS track** - Follow planned track, verify cross-track error
- [ ] **Compass / Heading** - Check against heading indicator and GPS
- [ ] **Flight instruments** - Cross-check attitude, altitude, airspeed

## Night / IMC Specific

- [ ] **Instrument lights** - Set (INST LTG switch)
- [ ] **Pitot Heat** - ON when required (PITOT HTR switch)
- [ ] **Anti-collision / Position lights** - As required

## Example Radio Calls

- **Position report** - "[ATC/Flight], UH-1H [Callsign], position [distance/bearing] from [fix], altitude [xxx] ft, estimating [point] at [time]."
- **Radio check** - "[Station], UH-1H [Callsign], radio check."

> **Tip:** Always cross-check independent navigation sources (GPS, VOR, ADF, Compass) and follow published procedures for IFR/low-visibility ops.
