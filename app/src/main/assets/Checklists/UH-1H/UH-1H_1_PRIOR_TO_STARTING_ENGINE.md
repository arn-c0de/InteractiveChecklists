# UH-1H Prior to Starting Engine

## Crew Coordination

- [ ] **Crew briefing** - Complete (mission, weather, threats, fuel, ROE)
- [ ] **Seat positions** - Pilot, Co-pilot/crew
- [ ] **ICS (Intercom System)** - Volume set, check communication between stations

## Overhead / Power / Electrical

- [ ] **Overhead switches and circuit breakers**
  - [ ] **DC circuit breakers** - IN except for armament and special equipment
- [ ] **EXT LTS switches** - Set as required
  - [ ] **ANTI COLL** - ON
  - [ ] **POSITION lights** - As required
- [ ] **MISC switches** - Set as follows:
  - [ ] **CARGO REL** - OFF
  - [ ] **WIPERS** - OFF
  - [ ] **CABIN HEATING** - OFF
  - [ ] **INST LTG** - As required
- [ ] **AC POWER controls** - Set as follows:
  - [ ] **PHASE** - AC
  - [ ] **INVTR** - OFF
- [ ] **DC POWER controls** - Set as follows:
  - [ ] **MAIN GEN** - ON
  - [ ] **VM selector** - ESS BUS
  - [ ] **NON-ESS BUS** - As required
  - [ ] **STARTER GEN** - START
  - [ ] **BAT** - ON

## Fire & Caution

- [ ] **FIRE warning light** - Test
- [ ] **Caution panel lights** - Test and REST

## Center Pedestal

- [ ] **Avionics equipment** - Set as desired
- [ ] **External stores jettison handle** - Check
- [ ] **DISP CONTROL panel** - Check
- [ ] **GOV** - AUTO
- [ ] **DE-ICE** - OFF
- [ ] **FUEL switches** - ON
  - [ ] **FUEL** - ON
  - [ ] **START FUEL** - ON
- [ ] **HYD CONT** - ON
- [ ] **FORCE TRIM** - ON
- [ ] **CHIP DET** - BOTH

## Flight Controls & Instruments

- [ ] **Flight controls** - Check free and correct movement of cyclic and collective
- [ ] **Altimeters** - Set to field elevation

## Example Radio Calls

- **Ground / Ramp (startup request)** - "Ground, UH-1H [Callsign], request engine start and taxi to parking/taxiway."
- **Start-clear call** - "UH-1H [Callsign] starting engines."

> **Note:** Complete items above before initiating engine start.
