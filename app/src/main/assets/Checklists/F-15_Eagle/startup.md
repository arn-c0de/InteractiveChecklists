# F-15C Eagle Cold Start Checklist (Simplified)

This checklist is for the F-15C Eagle in DCS World, which is a low-fidelity module. Procedures are simplified.

## Cold Start

- [ ] **Electrical Power ON:** Right Shift + L
- [ ] **Engine Start (Left):** Right Shift + Home (or assigned HOTAS button)
  - Monitor RPM.
- [ ] **Engine Start (Right):** Right Ctrl + Home (or assigned HOTAS button)
  - Monitor RPM.
  - Wait for both engines to spool up to stable RPM (around 67%).
- [ ] **Close Canopy:** Left Ctrl + C
- [ ] **Trim for Takeoff:** Left Alt + T
- [ ] **Radar ON:** I
- [ ] **HUD ON:** (Usually automatic or a brightness knob in the cockpit, but often simplified in FC3).
- [ ] **Master Mode:** NAV (usually default or set via a keybind).
- [ ] **Flight Controls:** Check full and free movement.
- [ ] **Taxi Clearance:** Request from ATC (usually `\` key, then F5, F1, F1).
- [ ] **Taxi Lights:** Right Alt + L (for gear lights, often used as taxi lights).
- [ ] **Wheel Brakes:** Release (W key).
- [ ] **Nose Wheel Steering:** Engage for tighter turns (S key).
- [ ] **Takeoff Flaps:** F
- [ ] **Takeoff Clearance:** Request from ATC (usually `\` key, then F5, F1, F1).
