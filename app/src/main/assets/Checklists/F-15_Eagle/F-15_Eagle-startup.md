# F-15C Eagle Startup Checklist

## Prior to Engine Start

- [ ] **External Power** - Check available or battery power
- [ ] **Electrical Power** - ON
- [ ] **Parking Brake** - Set
- [ ] **Throttles** - OFF/IDLE
- [ ] **Flight Controls** - Check free movement
- [ ] **Canopy** - Open (for pre-flight, will close after start)

## Engine Start

- [ ] **Engine Start (Left)** - Start
  - Monitor left engine RPM increase
  - Wait for stable idle (approximately 67% RPM)
- [ ] **Left Engine Instruments** - Check (oil pressure, RPM, temperature)
- [ ] **Engine Start (Right)** - Start
  - Monitor right engine RPM increase
  - Wait for stable idle (approximately 67% RPM)
- [ ] **Right Engine Instruments** - Check (oil pressure, RPM, temperature)
- [ ] **Both Engines** - Verify stable at idle

## After Engine Start

- [ ] **Canopy** - Close and lock
- [ ] **Hydraulic Pressure** - Check
- [ ] **Flight Controls** - Check full and free movement
- [ ] **Trim** - Set for takeoff
- [ ] **Radar** - ON
- [ ] **HUD** - Check ON and brightness set
- [ ] **Master Mode** - NAV
- [ ] **INS/Navigation** - Check aligned
- [ ] **Fuel Quantity** - Check
- [ ] **Oxygen** - Check
- [ ] **Instruments** - Check all systems
