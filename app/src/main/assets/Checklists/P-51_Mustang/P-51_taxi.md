# P-51 Mustang Taxi Checklist

## Taxi Procedures

- [ ] **Wing Flaps:** UP
- [ ] **Oil and Coolant Shutters:** OPEN
- [ ] **Tail Wheel:**
    - [ ] For straight taxiing, pull the stick slightly back of neutral to lock the tail wheel. Ensure the tailwheel is straight before locking.
    - [ ] In the locked position, the tail wheel can be turned 6° right or left with the rudder pedals.
    - [ ] For sharp turns, push the stick forward of neutral to allow full swiveling action of the tail wheel.
    - [ ] If the tailwheel is locked when not straight, move forward with it unlocked to straighten it out.
- [ ] **Steering:**
    - [ ] Steer using toe brakes and throttling up.
    - [ ] Use individual wheel brakes for turning.
    - [ ] Steer a zig-zag course for better visibility.
- [ ] **Brakes:** Use brakes as little as possible.
