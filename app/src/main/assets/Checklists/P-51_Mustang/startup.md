# P-51 Mustang Startup Checklist

## Preflight Inspection

- [ ] Visual inspection - Complete
- [ ] Control surfaces - Check for freedom of movement
- [ ] Fuel quantity - Check
- [ ] Oil level - Check
- [ ] Landing gear - Inspect tires and struts
- [ ] Propeller - Check for nicks and damage
- [ ] Pitot tube - Remove cover, check clear

## Cockpit Preparation

- [ ] Seat and rudder pedals - Adjust
- [ ] Seat belt and shoulder harness - Fasten
- [ ] Circuit breakers - Check in
- [ ] Battery switch - ON
- [ ] Fuel selector - Main tank
- [ ] Fuel boost pump - ON
- [ ] Fuel pressure - Check (5-7 PSI)
- [ ] Carburetor air filter - COLD
- [ ] Supercharger - AUTO
- [ ] Propeller control - Full forward (High RPM)
- [ ] Throttle - Cracked open (1/2 inch)
- [ ] Mixture - IDLE CUT OFF

## Engine Start

- [ ] Fire extinguisher - Available
- [ ] Area clear - Call "CLEAR PROP"
- [ ] Primer - Pump 2-4 strokes
- [ ] Magnetos - BOTH
- [ ] Starter switch - ENGAGE
- [ ] Mixture - RICH (when engine fires)
- [ ] Oil pressure - Check rising (within 30 seconds)
- [ ] Throttle - 1000-1200 RPM
- [ ] Primer - Lock
- [ ] Generator - Check online
- [ ] Ammeter - Check positive charge
- [ ] Suction gauge - Check (4-5 inches Hg)

## Before Taxi

- [ ] Radio - Set frequencies
- [ ] Altimeter - Set to field elevation
- [ ] Flight instruments - Check and set
- [ ] Directional gyro - Set to compass
- [ ] Elevator trim - Set for takeoff
- [ ] Rudder trim - Neutral
- [ ] Aileron trim - Neutral
- [ ] Fuel boost pump - ON
- [ ] Carburetor heat - COLD
- [ ] Coolant flaps - As required
- [ ] Oil cooler flaps - As required
- [ ] Brakes - Test

## Taxi

- [ ] Release parking brake
- [ ] Throttle - Advance smoothly
- [ ] Brakes - Test while taxiing
- [ ] Flight controls - Free and correct
- [ ] Engine instruments - Monitor (especially coolant temp)
- [ ] Rudder - S-turns for visibility

## Engine Runup

- [ ] Parking brake - Set
- [ ] Brakes - Hold
- [ ] Throttle - 1700 RPM
- [ ] Magnetos - Check both (drop not to exceed 100 RPM)
- [ ] Propeller - Cycle (RPM increase/decrease)
- [ ] Engine instruments - Check all green
- [ ] Carburetor heat - Test (RPM drop expected)
- [ ] Throttle - Return to idle (800-1000 RPM)

## Before Takeoff

- [ ] Flight controls - Free and correct
- [ ] Elevator trim - Set for takeoff (slightly nose up)
- [ ] Rudder trim - Set (right rudder for takeoff)
- [ ] Fuel - Main tank, boost pump ON
- [ ] Mixture - RICH
- [ ] Propeller - Full forward (High RPM)
- [ ] Coolant flaps - OPEN
- [ ] Oil cooler flaps - OPEN
- [ ] Canopy - Locked
- [ ] Harness - Tight and secure
- [ ] Instruments - Final check
- [ ] Clearance - Obtained

## Takeoff

- [ ] Brakes - Hold
- [ ] Throttle - Full power smoothly
- [ ] Engine instruments - Check all green
- [ ] Brakes - Release
- [ ] Rudder - Right rudder as needed (counteract torque)
- [ ] Speed - Monitor
- [ ] Rotate - 90-100 MPH
- [ ] Climb speed - 120-140 MPH
- [ ] Landing gear - UP (after positive rate)
- [ ] Coolant flaps - As required
- [ ] Oil cooler flaps - As required

## After Takeoff / Climb

- [ ] Airspeed - Maintain 150-170 MPH
- [ ] Throttle - 40-46 inches Hg
- [ ] RPM - 2700
- [ ] Mixture - As required for altitude
- [ ] Fuel boost pump - OFF
- [ ] Engine instruments - Monitor
- [ ] Coolant temp - 100-120°C
- [ ] Oil temp - 70-90°C
- [ ] Oil pressure - 70-90 PSI
