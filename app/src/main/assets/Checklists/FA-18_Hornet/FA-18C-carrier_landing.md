# FA-18 Hornet Carrier Landing Checklist

## Case I Recovery (Day VMC)

### Initial Approach

- [ ] ATIS - Obtain
- [ ] Altimeter - Set 29.92 (standard)
- [ ] Fuel state - Check (3000+ lbs for single approach)
- [ ] Configuration - Clean
- [ ] Speed - 350 KIAS
- [ ] Charlie call - Made at 10 nm
- [ ] Marshall holding - If required
- [ ] Approach clearance - Obtain
- [ ] Break altitude - 800' AGL at 3 nm

### The Break

- [ ] Initial - Cross at 800' AGL, 350 KIAS
- [ ] Smoke - Check visibility for visual reference
- [ ] Break - At abeam position, 60-80° bank
- [ ] Throttles - Idle
- [ ] Speedbrake - Extend
- [ ] G-loading - 3-4 G in break
- [ ] Landing gear - DOWN (below 300 KIAS)
- [ ] Landing gear lights - Three green
- [ ] Flaps - FULL
- [ ] Hook - DOWN
- [ ] Roll out - 180° heading change
- [ ] Downwind - Stabilize at 600' AGL, 150 KIAS

### Downwind

- [ ] Altitude - 600' AGL
- [ ] Airspeed - 150 KIAS
- [ ] Configuration - Gear down, flaps full, hook down
- [ ] Landing checks - Complete
- [ ] Abeam position - Note when abeam stern
- [ ] Interval - Check spacing from preceding aircraft
- [ ] Fuel - Monitor (min 2200 lbs in groove)

### The Turn to Final

- [ ] Position - 15-20 seconds past abeam
- [ ] Bank - 30° coordinated turn
- [ ] Throttles - Adjust to maintain altitude
- [ ] Airspeed - Maintain 150 KIAS
- [ ] Altitude - Maintain 600' AGL
- [ ] Rate of descent - Start descent approaching 90°
- [ ] Roll out - Aligned with centerline

### Groove / Final Approach

- [ ] Lineup - Centered on angle deck
- [ ] Ball call - "Hornet, ball, [fuel state]"
- [ ] Meatball - Acquire and reference
- [ ] Lineup - Reference datum lights
- [ ] AOA - On speed (15 units)
- [ ] Power - Small throttle corrections
- [ ] Ball - Fly the ball (slight high initially)
- [ ] Lineup - Maintain centerline
- [ ] Scan - Ball, lineup, angle of attack
- [ ] Power - Smooth corrections only
- [ ] LSO calls - Listen and respond

### At The Ramp

- [ ] Meatball - Slightly high
- [ ] Power - On, not idle
- [ ] AOA - On speed (amber donut)
- [ ] Lineup - Centered
- [ ] Touchdown - Aim for 3-wire
- [ ] Throttles - Full power at touchdown
- [ ] Stick - Hold attitude, don't push forward

### Arrested Landing

- [ ] Throttles - Full power (already applied)
- [ ] Stick - Neutral, don't fight decel
- [ ] Brakes - Do NOT apply
- [ ] Wait for stop - Let arresting gear do work
- [ ] Throttles - Idle after complete stop
- [ ] Hook - Check trapped (deceleration felt)

### Trap (After Arrested Landing)

- [ ] Throttles - Idle
- [ ] Brakes - Hold
- [ ] Yellow shirt signals - Acknowledge
- [ ] Tension - Wait for wire tension release
- [ ] Throttle - As directed for taxi
- [ ] Hook - UP (when signaled)
- [ ] Fold wings - As directed
- [ ] Taxi - Follow yellow shirt
- [ ] Parking - Shut down per procedures

## Case II Recovery (Night or IMC)

### Approach

- [ ] Marshall - Hold at assigned altitude
- [ ] Approach clearance - Obtain
- [ ] Altimeter - Set ship's QNH
- [ ] Descent - Begin at assigned time
- [ ] TACAN - Monitor bearing/distance
- [ ] Radar vectors - Follow controllers
- [ ] Speed - 250 KIAS until platform
- [ ] Platform - Level at 1200' (5 nm)

### Platform

- [ ] Altitude - 1200' MSL
- [ ] Distance - 5 nm from ship
- [ ] Speed - 250 KIAS
- [ ] Configuration - Dirty up
- [ ] Landing gear - DOWN
- [ ] Flaps - FULL
- [ ] Hook - DOWN
- [ ] Landing checks - Complete
- [ ] Speed - Slow to 150 KIAS
- [ ] Turn to final - At 3 nm

### Final Approach (Instrument)

- [ ] Heading - Fly ship's heading
- [ ] ILS - Intercept glideslope
- [ ] ICLS - Monitor needles
- [ ] AOA - Maintain on-speed
- [ ] Descent rate - 600-700 FPM
- [ ] Altitude calls - From altimeter
- [ ] Needles - Fly centered
- [ ] Ball call - At 3/4 mile
- [ ] Transition to visual - Acquire ball

### Visual Transition (3/4 Mile)

- [ ] Meatball - Acquire visually
- [ ] ILS/ICLS - Reference but fly ball
- [ ] AOA - On speed
- [ ] Lineup - Center on angle deck
- [ ] Ball - Primary reference
- [ ] Power - Small corrections
- [ ] Continue - As Case I final

## Case III Recovery (Night or Poor Weather)

### Approach

- [ ] Marshall - Hold at assigned altitude
- [ ] Approach clearance - Obtain with ACLS
- [ ] ACLS - Mode 1 or Mode 2
- [ ] Autopilot - Engage as briefed
- [ ] Speed - As assigned
- [ ] Configuration - Dirty up
- [ ] Landing gear - DOWN
- [ ] Flaps - FULL
- [ ] Hook - DOWN
- [ ] ACLS - Monitor fly-to needles

### ACLS Approach

- [ ] Autopilot - Monitor or hand fly
- [ ] Needles - Keep centered
- [ ] AOA - On speed
- [ ] Altitude - Monitor crossing
- [ ] Ball call - Make at 3/4 mile if visual
- [ ] Continue approach - Trust instruments
- [ ] Throttle - Small corrections
- [ ] Transition visual - Acquire ball when able

## Bolter / Waveoff

### Bolter (Missed Arrestment)

- [ ] Throttles - Already at full power
- [ ] Rotate - 10° nose up
- [ ] Positive rate - Confirm climbing
- [ ] Landing gear - UP
- [ ] Flaps - HALF
- [ ] Hook - UP
- [ ] Tower - Report bolter
- [ ] Upwind - Climb straight ahead
- [ ] Speed - Accelerate to 250 KIAS
- [ ] Pattern - Enter downwind or as directed

### Waveoff (LSO Directed)

- [ ] Throttles - Full power immediately
- [ ] Rotate - 10° nose up
- [ ] Positive rate - Confirm
- [ ] Landing gear - UP
- [ ] Flaps - HALF
- [ ] Hook - UP
- [ ] Continue - As directed by tower
- [ ] Pattern re-entry - As assigned

## Emergency Procedures

### One Wire (Barricade)

- [ ] Emergency declared - Advise tower
- [ ] Barricade rigged - Wait for confirmation
- [ ] Approach - Normal until in close
- [ ] AOA - Slightly fast (14 units)
- [ ] Aim point - Centered in barricade
- [ ] Touchdown - Fly into barricade
- [ ] Throttles - Idle at contact
- [ ] Egress - Immediate after stop

### Field Arrestment (Divert)

- [ ] Emergency declared - If applicable
- [ ] Field approach - Standard approach
- [ ] Hook - DOWN
- [ ] Gear cable - Verify in place
- [ ] Touchdown - Normal
- [ ] Engage cable - Let hook catch
- [ ] Throttles - Idle
- [ ] Stop - Wait for complete stop
