# FA-18 Hornet Landing Checklist

## Field Landing Pattern

The typical field landing pattern is a left-hand circuit entered from a 45-degree downwind leg.

- **Downwind Leg:** Fly at pattern altitude (typically 1500 ft AGL) and configure the aircraft for landing.
- **Base Leg:** Turn 90 degrees left. Start a gradual descent.
- **Final Approach:** Turn another 90 degrees left to align with the runway. Continue the descent on a stable approach path.
- **Touchdown:** Aim for the touchdown zone of the runway.

## Before Landing - Ashore

- [ ] **ATC** - Obtain Landing Clearance
- [ ] **Master Arm Switch** - SAFE
- [ ] **ECM** - STBY

### Landing Checklist

- [ ] **LDG GEAR** - DN (Down)
- [ ] **FLAP SW** - FULL
- [ ] **HOOK** - AS REQ'D (Typically UP for field landing unless emergency)
- [ ] **ANTI SKID** - ON
- [ ] **HARNESS** - LOCKED & TIGHT
- [ ] **DISPENSER** - CHECK OFF

- [ ] **LDG/TAXI LIGHT SW** - ON

---

## After Landing

- [ ] **SPEEDBRAKE** - CLOSED
- [ ] **FLAPS** - AUTO
- [ ] **EXT LT PANEL** - SET
- [ ] **LDG/TAXI LT SW** - OFF
- [ ] **SEAT** - SAFE
- [ ] **T/O TRIM** - PUSH
- [ ] **OXY FLOW** - OFF
- [ ] **CANOPY** - FULL UP OR FULL DN

---

## Engine Shutdown

### Before Engine Shutdown
- [ ] **PARK BRK** - SET
- [ ] **INS** - OFF* (10 Sec before ENG shutdown)
- [ ] **SARI** - CAGE/LOCK
- [ ] **FLIR** - OFF
- [ ] **RADAR** - OFF
- [ ] **RWR** - PWR OFF
- [ ] **AVIONICS** - OFF
- [ ] **COMMS** - OFF
- [ ] **EXT AND INT LTS** - OFF
- [ ] **CRYPTO SW** - AS REQ'D
- [ ] **CANOPY** - OPEN

### Shutdown Sequence
- [ ] **BRAKE GAUGE** - Check 3,000 PSI
- [ ] **NWS** - DISENGAGE
- [ ] **FLAP SW** - FULL
> **NOTE:** Before engine shutdown, operate at Flight Idle or Ground Idle for 5 minutes.
- [ ] **L THROTTLE** - OFF
- [ ] **L HYD PRESS** - Check < 1,500
- [ ] **STICK** - Waggle Fore/Aft ±1"
- [ ] **L HYD PRESS** - Check < 800
- [ ] **FCS PAGE** - CHECK BLIN / Xs*
> *Code 63 or 67 / Xs in Rudder, Aileron or LEF = Fault.
- [ ] **DDIS/MPCD/HUD** - OFF
- [ ] **R THROTTLE** - OFF
- [ ] **BATT SW** - OFF*
> *When amber Flaps Lt illuminates.
