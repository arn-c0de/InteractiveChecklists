# FA-18 Hornet Air-to-Air Combat Checklist

- [ ] MASTER ARM switch (MASTER ARM) - ARM
- [ ] Radar - A/A mode selected
- [ ] Radar elevation - Set for scan volume
- [ ] RWR - Check operational
- [ ] CMDS - Check loaded and operational
- [ ] Datalink - Check operational
- [ ] IFF - MODE 4 on
- [ ] Fuel state - Check and monitor
- [ ] Weapons status - Verify on stores page
- [ ] AIM-120 - Check status
- [ ] AIM-9 - Check status
- [ ] Gun - 570 rounds, verify armed
- [ ] G-suit - Connected
- [ ] Oxygen - 100%
