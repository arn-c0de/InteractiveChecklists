# F/A-18 Hornet Conventional Airport Landing Checklist

## I. Approach Setup / Before Landing Checks
- [ ] **ATC Contact:** Contact Air Traffic Control for landing clearance.
- [ ] **Altimeter Setting:** Check and set altimeter.
- [ ] **Approach Clearance:** Obtain approach clearance (for shore operations).
- [ ] **IFF:** Set IFF as directed.
- [ ] **Weapons/Sensors:** Ensure weapons/sensors are safe/off as required.
- [ ] **Exterior Lights:** Set exterior lights for landing.
- [ ] **Radar:** Set RDR knob to STBY (no later than 10 NM from airfield).
- [ ] **ILS:** Turn ILS ON and set channel if required.
- [ ] **Hook Bypass:** Set HOOK BYPASS to FIELD (for airfield landing).
- [ ] **Anti-Skid:** Ensure Anti-Skid is ON for airfield landing.
- [ ] **Harness:** Harness locked.
- [ ] **Dispensers:** Dispensers set to safe.
- [ ] **Aircraft Weight:** Note aircraft weight; max landing weight for F/A-18C is 34,000 lbs.

## II. Traffic Pattern Entry / Downwind Leg
- [ ] **Initial Approach:** Fly at 800 ft AGL, offset to the right of the runway at approximately 350 knots (Navy-style approach).
- [ ] **Break:** Initiate a level break (turn to downwind) with idle throttle and speed brakes out.
- [ ] As airspeed decreases through 250 knots:
- [ ] **Landing Gear:** Lower landing gear.
- [ ] **Flaps:** Extend flaps to FULL.
- [ ] **Speed Brake:** Retract speed brake.
- [ ] **Descent to Downwind:** Descend to 600 ft AGL on the downwind leg.
- [ ] **Trim:** Trim aircraft to achieve on-speed Angle of Attack (AOA).
- [ ] **Landing Checklist Completion:** Complete the landing checklist once established on the downwind leg.
    - [ ] Verify: "3 DOWN AND LOCKED" (landing gear), "FLAPS FULL", "AOA CROSS-CHECKED".

## III. Final Approach
- [ ] **Glideslope:** Maintain a 2.5-3 degree glideslope.
- [ ] **Throttle Adjustments:** Constantly adjust throttles to maintain desired flight path and AOA. Small, precise throttle movements are key.
- [ ] **Descent Rate:** Adjust thrust to manage descent rate, avoiding excessive pitch changes with the stick.

## IV. Touchdown
- [ ] **Throttle:** Reduce throttles to idle at touchdown.
- [ ] **Drag:** Bring the stick full aft to increase drag from horizontal stabilizers.
- [ ] **Braking:** Gently apply wheel brakes once below approximately 100 knots indicated airspeed, using rudder pedals for directional control.
- [ ] **Nosewheel:** Once the nosewheel is on the ground, hold the stick full aft and brake lightly.

## V. After Landing Checks
- [ ] **Exit Runway:** Exit the runway to a taxiway.
- [ ] **Strobe/Landing Lights:** Turn off strobe and landing lights.
- [ ] **Flaps:** Retract flaps.
- [ ] **Hook:** Hook UP.
- [ ] **NWS:** Engage Nose Wheel Steering (NWS) as required for taxi.
