# F-16C Viper Takeoff Checklist

## Before Takeoff

- [ ] **Flight Controls** – Check (Full, Free, Correct movement)
- [ ] **Trim** – Set (Takeoff Trim)
- [ ] **Flaps** – Set (Takeoff/Auto)
- [ ] **Parking Brake** – Set
- [ ] **Canopy** – Closed, Locked, Light Off
- [ ] **Ejection Seat** – Armed
- [ ] **Oxygen** – On, 100%
- [ ] **Altimeter** – Set
- [ ] **HUD** – Set (ICP SYM rotary)
- [ ] **IFF** – As desired
- [ ] **Master Arm** – OFF
- [ ] **LASER ARM switch** – OFF
- [ ] **RF Switch** – NORM
- [ ] **SNSR PWR switches (4)** – OFF
- [ ] **DBU** – Check
- [ ] **Standby Altitude Indicator** – Set
- [ ] **FUEL QTY SEL knob** – NORM
- [ ] **EXT FUEL TRANS switch** – NORM
- [ ] **INSTR MODE knob** – As desired
- [ ] **ENG CONT switch** – PRI
- [ ] **JFS switch** – OFF
- [ ] **OIL pressure** – 15 psi (minimum)
- [ ] **HYD PRESS A & B** – 2850-3250 psi
- [ ] **Warning/Caution Lights** – Check (All clear)
- [ ] **Exterior Lights** – As required

## Line Up

- [ ] **Nose Wheel Steering** – Engaged
- [ ] **Brakes** – Hold
- [ ] **Throttle** – MIL Power (Check engine instruments for normal operation)
- [ ] **Brakes** – Release
- [ ] **Throttle** – MAX A/B (As required for takeoff performance)

## Takeoff Roll

- [ ] **Maintain Centerline**
- [ ] **Engine Instruments** – Check (Normal indications)
- [ ] **Rotation Speed** – Rotate
- [ ] **Liftoff** – Positive Rate of Climb
- [ ] **Landing Gear** – Up
- [ ] **Flaps** – Auto (if not already)
