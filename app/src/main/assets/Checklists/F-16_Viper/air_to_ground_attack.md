# F-16 Air-to-Ground Attack Checklist

## Pre-Attack Planning

- [ ] Target coordinates - Verify
- [ ] Target elevation - Note
- [ ] IP (Initial Point) - Identify
- [ ] Attack heading - Plan
- [ ] Weapons selected - Verify loadout
- [ ] Attack profile - Brief (dive, level, pop-up)
- [ ] Safe escape heading - Plan
- [ ] Threats - Identify SAM/AAA locations
- [ ] SEAD support - Coordinate if available
- [ ] Time on target - Verify
- [ ] Weather at target - Check
- [ ] Minimum altitude - Brief

## Weapons Setup

- [ ] Master arm - ARM
- [ ] SMS page - Display on MFD
- [ ] Weapons selector - Select appropriate weapon
- [ ] Release parameters - Set
- [ ] Fuze settings - Verify
- [ ] Delivery mode - Select (CCIP, CCRP, DTOS)
- [ ] Quantity - Set
- [ ] Interval - Set if multiple weapons

## CCIP Bombing (Continuously Computed Impact Point)

- [ ] Master arm - ARM
- [ ] Weapons - Select bombs
- [ ] Delivery mode - CCIP
- [ ] Target - Visually acquire
- [ ] Dive angle - 10-45° typical
- [ ] Airspeed - 400-500 KIAS recommended
- [ ] Altitude - Respect minimum (typically 3000-5000' AGL)
- [ ] HUD - Pipper on target
- [ ] Release - Pickle button when pipper on target
- [ ] Pull up - Smooth 4-5 G pull
- [ ] Egress - Execute safe escape maneuver
- [ ] BDA - Observe if able

## CCRP Bombing (Continuously Computed Release Point)

- [ ] Master arm - ARM
- [ ] Weapons - Select bombs
- [ ] Delivery mode - CCRP
- [ ] Target - Enter coordinates or designate
- [ ] Altitude - Maintain release altitude
- [ ] Airspeed - Maintain release speed
- [ ] Steering cues - Follow on HUD
- [ ] Release cue - Weapon auto-releases
- [ ] Continue - Maintain heading until release
- [ ] Egress - Execute after release

## DTOS (Dive Toss) Bombing

- [ ] Master arm - ARM
- [ ] Weapons - Select bombs
- [ ] Delivery mode - DTOS
- [ ] Target - Designate
- [ ] Dive angle - 20-30° typical
- [ ] Pull cue - HUD displays pull cue
- [ ] Pull up - When cue appears (5-6 G)
- [ ] Auto release - Weapon releases in climb
- [ ] Continue pull - Complete pull to safe altitude
- [ ] Egress - Turn to safe heading

## AGM-65 Maverick Employment

- [ ] Master arm - ARM
- [ ] Weapons selector - AGM-65
- [ ] Targeting pod - Slew to target (or HMCS)
- [ ] Maverick seeker - Uncage
- [ ] SOI - Select Maverick as Sensor of Interest
- [ ] Target lock - Achieve good contrast lock
- [ ] Range - Check within parameters (max ~15 nm)
- [ ] In range - Verify in-range cue
- [ ] Launch - Pickle when all cues satisfied
- [ ] Track - Monitor missile flight
- [ ] BDA - Assess results
- [ ] RIFLE call - Make radio call on launch

## Laser Guided Bomb (LGB) Employment

- [ ] Master arm - ARM
- [ ] Weapons - Select LGB
- [ ] Targeting pod - Power on
- [ ] Target - Acquire in targeting pod
- [ ] Laser code - Verify matches bomb
- [ ] Point track - Establish on target
- [ ] Laser - ARM
- [ ] Delivery - As briefed (dive or level)
- [ ] Release - Pickle when in parameters
- [ ] Laser firing - Verify laser is firing
- [ ] Continue - Maintain laser until impact
- [ ] BDA - Assess results
- [ ] Laser safe - After impact

## JDAM Employment (GPS Guided)

- [ ] Master arm - ARM
- [ ] Weapons - Select JDAM
- [ ] GPS - Check good signal quality
- [ ] Target coordinates - Verify programmed
- [ ] Delivery mode - CCRP typically
- [ ] Release altitude - Check minimums
- [ ] Release parameters - Verify in parameters
- [ ] Steering - Follow HUD cues
- [ ] Release - Auto or manual pickle
- [ ] Egress - Immediate after release
- [ ] BDA - If able to observe

## Strafing Run (20mm Gun)

- [ ] Master arm - ARM
- [ ] Weapons selector - GUN
- [ ] Target - Visually acquire
- [ ] Dive angle - 10-30° typical
- [ ] Airspeed - 350-450 KIAS
- [ ] Altitude - Minimum 2000' AGL breakoff
- [ ] HUD - Funnel or EEGS pipper on target
- [ ] Range - 4000-6000 feet optimal
- [ ] Trigger - Squeeze for 1-2 second burst
- [ ] Pull up - At minimum altitude (4-5 G)
- [ ] Egress - Safe heading after pull
- [ ] BDA - Assess if able

## Rocket Employment (If equipped)

- [ ] Master arm - ARM
- [ ] Weapons - Select rockets
- [ ] Delivery - CCIP mode
- [ ] Dive angle - 10-20° typical
- [ ] Airspeed - 350-450 KIAS
- [ ] Range - 1-2 nm typical
- [ ] Pipper - Place on target
- [ ] Release - Pickle (single or ripple)
- [ ] Pull up - At safe altitude
- [ ] Egress - Safe heading

## Pop-Up Attack

- [ ] Low altitude ingress - 500-1000' AGL
- [ ] IP - Cross at briefed altitude/speed
- [ ] Pop point - Identify
- [ ] Pull up - Smooth 4-5 G pull to 10-15°
- [ ] Roll - To attack heading (60-90° bank)
- [ ] Acquire target - Visual or sensors
- [ ] Dive - Roll to dive (30-45° typical)
- [ ] Weapons delivery - Execute as briefed
- [ ] Egress - Low altitude safe heading

## Threat Reaction

- [ ] RWR - Monitor continuously
- [ ] SAM launch - React immediately
- [ ] Chaff program - Deploy
- [ ] Break - Hard turn to defeat
- [ ] Notch - If radar guided
- [ ] Flares - If IR threat
- [ ] Jink - Unpredictable maneuvering
- [ ] Abort attack - If required for survival
- [ ] SEAD support - Request if available

## Post-Attack

- [ ] Master arm - SAFE
- [ ] Egress - Execute safe escape
- [ ] Altitude - Climb if threat allows
- [ ] Speed - Accelerate to cruise
- [ ] Fuel check - Monitor state
- [ ] BDA - Report results
- [ ] Weapons - Check remaining
- [ ] Systems - Check for damage/malfunctions
- [ ] Rejoin - With flight if applicable
- [ ] RTB or next target - As briefed
