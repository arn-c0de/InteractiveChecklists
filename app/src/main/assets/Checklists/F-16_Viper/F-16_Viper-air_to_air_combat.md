# F-16 Air-to-Air Combat Checklist

## Pre-Combat Setup

- [ ] MASTER ARM switch (MASTER ARM) - ARM
- [ ] Radar - A/A mode
- [ ] RWR - Check operational
- [ ] CMDS programs - Loaded and verified
- [ ] Fuel state - Check sufficient
- [ ] Weapons - Select appropriate A/A weapon
- [ ] Missiles - Check status (AIM-9, AIM-120)
- [ ] Gun - Armed if required
- [ ] G-suit - Connected and functioning
- [ ] Oxygen - 100%
- [ ] IFF - As required (MODE 4 on)
- [ ] HMCS - Check aligned
- [ ] Datalink - Check operational

## Radar Setup (A/A Mode)

- [ ] Radar mode - Select appropriate (RWS, TWS, STT)
- [ ] Scan volume - Set for expected threats
- [ ] Range - Set appropriate for situation
- [ ] PRF - AUTO or manual as required
- [ ] Altitude filter - Set if required
- [ ] Radar scope - Monitor for contacts
- [ ] Target acquisition - Lock priority targets
- [ ] IFF interrogation - Verify before engaging

## AIM-120 AMRAAM Employment

- [ ] Weapons selector - AIM-120
- [ ] Radar lock - STT on target
- [ ] Range - Within Rmax envelope
- [ ] DLZ display - Check shoot cues
- [ ] MASTER ARM switch (MASTER ARM) - ARM
- [ ] Target data - Verify on HUD
- [ ] Uncage - As required
- [ ] Fire - Weapon release button
- [ ] Guidance - Monitor missile flight
- [ ] Breaklock - When appropriate
- [ ] Crank - Execute for missile support
- [ ] Datalink - Monitor missile via datalink
- [ ] Second shot - Evaluate if required
- [ ] Defensive - Go defensive if required

## AIM-9 Sidewinder Employment

- [ ] Weapons selector - AIM-9
- [ ] MASTER ARM switch (MASTER ARM) - ARM
- [ ] Radar - May use or visual acquisition
- [ ] HMCS - Use for off-boresight targeting
- [ ] Seeker tone - Listen for good tone
- [ ] Target aspect - Within parameters
- [ ] Range - Within Rmax (typically < 10 nm)
- [ ] Launch - Weapon release when tone good
- [ ] Maneuver - Support missile flight path
- [ ] Track - Monitor missile flight
- [ ] Assess - Hit or miss evaluation

## Gun Employment (20mm M61A1)

- [ ] Weapons selector - GUN
- [ ] MASTER ARM switch (MASTER ARM) - ARM
- [ ] Radar - Acquire target (or visual)
- [ ] HUD - Funnel or EEGS symbology
- [ ] Range - Optimal 2000-4000 feet
- [ ] Aspect - Lag pursuit to lead pursuit
- [ ] G - Moderate to avoid high-G gun tracking
- [ ] Pipper - Place on target
- [ ] Trigger - Squeeze for burst (1-2 second)
- [ ] Assess - Check for hits/damage
- [ ] Disengage - If no hits, reposition

## Basic Fighter Maneuvers (BFM)

- [ ] Lift vector - Point at bandit
- [ ] Corner velocity - Maintain for best turn (330-440 KIAS)
- [ ] Energy - Manage altitude and airspeed
- [ ] Turn rate - Maximize when required
- [ ] Turn radius - Minimize when required
- [ ] High yo-yo - Execute when overtaking
- [ ] Low yo-yo - Execute to gain position
- [ ] Lag pursuit - For positional advantage
- [ ] Lead pursuit - For weapons solution
- [ ] Pure pursuit - To close range quickly
- [ ] Vertical fight - Use altitude advantage
- [ ] Horizontal fight - When energy state allows

## Defensive Maneuvers

- [ ] Tally bandit - Visually acquire threat
- [ ] RWR - Monitor threat warnings
- [ ] Break turn - Hard turn into threat (> 6 G)
- [ ] Chaff - Deploy (program 1-4 as briefed)
- [ ] Flares - Deploy for IR threats
- [ ] Beam - Turn to put threat on beam
- [ ] Notch - Defeat Doppler radar
- [ ] Jink - Random unpredictable maneuvers
- [ ] Vertical maneuver - Extend using vertical
- [ ] Energy - Manage for sustained defense
- [ ] Support - Call for wingman support
- [ ] Extend - Increase range when able

## Chaff/Flare Programs

- [ ] MAN 1 - Manual chaff burst
- [ ] MAN 2 - Manual flare burst
- [ ] MAN 3 - Manual chaff/flare mix
- [ ] MAN 4 - Manual salvo
- [ ] SEMI - Semi-automatic expendables
- [ ] AUTO - Automatic based on RWR
- [ ] Bypass - Override for manual control
- [ ] Inventory - Monitor remaining count

## Missile Defense (Active Radar)

- [ ] RWR warning - Detect launch
- [ ] Chaff - Deploy immediate program
- [ ] Break turn - Hard turn to defeat missile
- [ ] Beam - Put missile on beam
- [ ] Notch altitude - Get into ground clutter
- [ ] Chaff continued - Multiple bursts
- [ ] Energy management - Maintain corner velocity
- [ ] Assess - Monitor missile tracking

## Missile Defense (IR)

- [ ] Flares - Deploy immediate program
- [ ] Break turn - Hard turn to defeat seeker
- [ ] Sun - Use sun to break track if available
- [ ] Flares continued - Multiple bursts
- [ ] Reduce IR signature - Throttle management
- [ ] Energy management - Maintain maneuverability

## Engagement Priorities

- [ ] Targeting priority - Closest threat first
- [ ] Shot doctrine - BVR then WVR progression
- [ ] Fuel state - Monitor constantly
- [ ] Weapons state - Track remaining weapons
- [ ] Bingo fuel - Know and respect
- [ ] Winchester - Call when out of weapons
- [ ] Tactical situation - Reassess continuously

## Communication

- [ ] BOGEY call - Unknown radar contact
- [ ] BANDIT call - Confirmed hostile
- [ ] TALLY call - Visual on bandit
- [ ] NO JOY - No visual on bandit
- [ ] CONTACT call - Radar lock
- [ ] CAPTURED call - STT lock
- [ ] FOX 1 - AIM-7 launch (if applicable)
- [ ] FOX 2 - AIM-9 launch
- [ ] FOX 3 - AIM-120 launch
- [ ] RIFLE - AGM-65 launch (A/G)
- [ ] GUNS GUNS - 20mm gun engaged
- [ ] BREAK - Urgent defensive turn required
- [ ] DEFENDING - Engaged defensively
- [ ] EXTENDING - Disengaging to extend range

## Post-Engagement

- [ ] BDA - Battle damage assessment
- [ ] Fuel state - Check and report
- [ ] Weapons remaining - Check and report
- [ ] Aircraft status - Check systems
- [ ] Rejoin - Execute rejoin procedures
- [ ] RTB decision - Evaluate mission completion
- [ ] Debrief items - Mental note for debrief
