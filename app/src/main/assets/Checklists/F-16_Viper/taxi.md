# F-16C Viper Taxi Checklist

## Taxi Procedures

- [ ] **Canopy** – Down and locked.
- [ ] **Altimeter (Electric)** – Matches HUD.
- [ ] **Altimeter (Servo Pneumatic)** – Within 75 ft of Electric.
- [ ] **Air Pressure** – Set (e.g., 29.92 in mercury).
- [ ] **Exterior Lights** – Set as desired.
- [ ] **EGI Knob** – NAV.
- [ ] **Nose Wheel Steering (NWS)** – Engage/ON.
- [ ] **Flight Controls** – Cycle and check for full, free movement of rudder, stick, and surfaces.
- [ ] **Flaps** – Normal.
- [ ] **Trim Needles** – Centered.
- [ ] **Engine** – Set to PRI.
- [ ] **Air Brakes** – Closed.
- [ ] **Fuel Panel** – Normal.
- [ ] **FTI** – Below 650C.
- [ ] **Oxygen** – ON.
- [ ] **Fuel Quantity Select Knob** – NORMAL.
- [ ] **Oil Pressure** – Between 15 and 65 PSI.
- [ ] **Caution, Warning, and Indication Lights** – All OFF.
- [ ] **Ejection Safety Lever** – Armed.
- [ ] **Throttle** – Just above idle for movement.
- [ ] **HSI** – Check turning.
- [ ] **Magnetic Heading Tape (HUD)** – Check changing.
- [ ] **Wheel Brakes** – Use as needed; avoid riding to prevent overheating.
