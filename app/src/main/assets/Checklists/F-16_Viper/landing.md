# F-16 Landing Procedures

## Before Descent / Approach Planning

- [ ] ATIS - Obtain current
- [ ] Weather - Review (winds, visibility, ceiling)
- [ ] Runway - Identify active
- [ ] Approach type - Brief (visual, ILS, TACAN, etc.)
- [ ] Minimums - Review
- [ ] Fuel state - Check sufficient plus reserves
- [ ] Approach plate - Review
- [ ] Missed approach - Brief
- [ ] Emergency procedures - Review

## Initial Approach

- [ ] ATC clearance - Obtain approach clearance
- [ ] Altimeter - Set QNH
- [ ] Course - Set in HSI/MFD
- [ ] ILS frequency - Set if applicable
- [ ] TACAN - Set if applicable
- [ ] Speed - Reduce to 300 KIAS or as assigned
- [ ] Fuel quantity - Note for landing calculations
- [ ] Configuration - Clean until on approach
- [ ] Radar altimeter - Set minimum altitude

## Descent

- [ ] Throttle - Reduce for descent
- [ ] Speed - Maintain 300 KIAS or as assigned
- [ ] Descent rate - 2000-4000 FPM typical
- [ ] Altitude - Cross restrictions as assigned
- [ ] Speed - Comply with restrictions (250 KIAS below 10,000')
- [ ] TCAS - Monitor traffic
- [ ] Trim - Adjust for descent attitude

## Overhead Pattern Entry (If Applicable)

- [ ] Initial point - Cross at 1500' AGL, 300 KIAS
- [ ] Pitch out - Execute at break point
- [ ] G-load - Smooth 4-5 G turn
- [ ] Throttle - Idle
- [ ] Speedbrake - Extend
- [ ] Landing gear - DOWN (when below 300 KIAS)
- [ ] Landing gear lights - Three green
- [ ] Downwind - Roll out, 180 KIAS
- [ ] Base turn - 90° heading change at appropriate point
- [ ] Final turn - Roll out aligned with runway

## Pre-Landing Configuration

- [ ] Speed - Reduce to approach speed
- [ ] Landing gear lever - DOWN
- [ ] Landing gear lights - Three green (verify)
- [ ] Stores - Safe
- [ ] Master arm - SAFE
- [ ] Speedbrake - As required
- [ ] Trim - Adjust for approach configuration

## Final Approach

- [ ] Runway - Visually acquire and identify
- [ ] Lineup - Establish on centerline
- [ ] Glideslope - Intercept and maintain (typically 3°)
- [ ] Approach speed - Maintain computed approach speed
  - [ ] Light weight: 150-165 KIAS
  - [ ] Medium weight: 165-180 KIAS  
  - [ ] Heavy weight: 180-195 KIAS
  - [ ] Adjust for 1/2 headwind + gust
- [ ] Throttle - Modulate for speed control
- [ ] Pitch - Control with stick for glidepath
- [ ] Power - Control with throttle for speed
- [ ] Lineup - Maintain with rudder/aileron
- [ ] Angle of attack - Monitor (optimum 11-13° units)
- [ ] Altitude - Call out 500', 200', 100', 50'
- [ ] Speed - Maintain on speed throughout
- [ ] Trim - Minor adjustments only

## Short Final

- [ ] Runway - Positively identified
- [ ] PAPI/VASI - Check on glidepath (red over white)
- [ ] Lineup - Centered on runway
- [ ] Speed - On target approach speed
- [ ] Sink rate - 500-700 FPM typical
- [ ] Power - Smooth adjustments
- [ ] Crosswind correction - Maintain as needed
- [ ] Trim - Stable

## Touchdown

- [ ] Throttle - Idle (at 10-20' AGL)
- [ ] Flare - Slight nose up attitude increase
- [ ] Aim point - Main gear on numbers or touchdown zone
- [ ] Sink rate - Reduce to 100-300 FPM
- [ ] Drift - Zero at touchdown
- [ ] Main gear - Touch down first
- [ ] Nose gear - Lower smoothly after main gear
- [ ] Wings - Level at touchdown

## Rollout

- [ ] Throttle - Confirm idle
- [ ] Nosewheel - Lower to runway
- [ ] Nosewheel steering - Engaged automatically
- [ ] Brakes - Apply smoothly as needed
- [ ] Drag chute - Deploy if installed/required
- [ ] Rudder - Maintain directional control
- [ ] Speedbrake - Retract after nose gear down
- [ ] Speed - Monitor deceleration
- [ ] Runway - Maintain centerline
- [ ] Turnoff speed - Slow to appropriate speed (<50 knots)

## After Landing / Taxi

- [ ] Drag chute - Jettison if deployed
- [ ] Trim - Neutral
- [ ] Flaps - UP
- [ ] Speedbrake - Check retracted
- [ ] Lighting - Day/night taxi configuration
- [ ] IFF - STBY
- [ ] Radar - STBY
- [ ] Taxi speed - Safe speed (not exceed 25 knots turns)
- [ ] Brakes - Check operation
- [ ] Ground frequency - Contact if required

## Parking

- [ ] Parking spot - Identify and taxi to
- [ ] Chocks - Ground crew position
- [ ] Throttle - IDLE
- [ ] Parking brake - SET
- [ ] Hand signals - Acknowledge with ground crew
- [ ] Throttle - OFF
- [ ] Ejection seat - SAFE (install pins)
- [ ] Master arm - SAFE
- [ ] Avionics - OFF
- [ ] Generator - Monitor off line
- [ ] JFS - Check not running

## Shutdown

- [ ] Throttle - OFF position
- [ ] Engine - Monitor rundown
- [ ] Hydraulic pressure - Monitor decrease
- [ ] Canopy - Open when engine stopped
- [ ] Oxygen - Disconnect
- [ ] G-suit - Disconnect
- [ ] Seat belt and harness - Release
- [ ] Forms - Complete aircraft forms
- [ ] Debrief items - Note for maintenance
- [ ] Battery - OFF when ready to egress
- [ ] External power - Connect if required

## Go-Around / Missed Approach

- [ ] Throttle - MIL or AB as required
- [ ] Pitch - Climb attitude (10-15° nose up)
- [ ] Trim - Adjust for climb
- [ ] Positive rate - Verify
- [ ] Landing gear - UP
- [ ] Speedbrake - Retract
- [ ] ATC - Advise going around
- [ ] Missed approach procedure - Execute as published
- [ ] Speed - Accelerate to climb speed (250+ KIAS)
- [ ] Altitude - Comply with missed approach altitudes
- [ ] Configuration - Clean up as appropriate

## Emergency Landing

- [ ] Declare emergency - If required
- [ ] Gear - Attempt normal extension
- [ ] Emergency gear - Use if normal fails
- [ ] Barrier engagement - If required and available
- [ ] Approach speed - Adjust as briefed for emergency
- [ ] Landing - Plan for long rollout if no brakes
- [ ] Emergency equipment - Advise on final
- [ ] Egress - Prepare for immediate egress after stop
