# F-16 Takeoff Procedures

## Pre-Takeoff Final Items

- [ ] Canopy - Closed and locked
- [ ] Seat - ARMED
- [ ] Harness - Locked and tight
- [ ] Ejection seat pins - Removed and stowed
- [ ] Oxygen - 100%, flow check
- [ ] G-suit - Connected and check inflation
- [ ] HMCS - Aligned and operational
- [ ] DED takeoff page - Set and verified
- [ ] Trim - Set for takeoff (slightly nose up)
- [ ] Leading edge flaps - AUTO
- [ ] Stores configuration - Verified
- [ ] Master arm - As briefed (SAFE/SIMULATE/ARM)
- [ ] IFF - As required (NORM)
- [ ] Radar - STBY or as required
- [ ] Fuel quantity - Sufficient for mission
- [ ] Takeoff clearance - Obtained
- [ ] Runway heading - Confirmed

## Formation Takeoff (If Applicable)

- [ ] Formation lights - As briefed
- [ ] Spacing - Proper for element takeoff
- [ ] Hand signals - Acknowledge ready
- [ ] Power up - Simultaneous with lead

## Normal Takeoff (Clean Configuration)

- [ ] Brakes - Hold firmly
- [ ] Throttle - Advance smoothly to MIL power
- [ ] Engine instruments - Quick check
  - [ ] RPM - 100%
  - [ ] FTIT - Normal range (within limits)
  - [ ] Nozzle position - As expected
  - [ ] Hydraulic pressure - Normal (A&B)
  - [ ] Oil pressure - Normal
- [ ] Gauges - Green (no caution lights)
- [ ] Brakes - Release smoothly
- [ ] Rudder - Use as needed to maintain centerline
- [ ] Nosewheel steering - Maintain directional control
- [ ] Airspeed - Call 80 knots, 100 knots
- [ ] Decision speed - Check (V1 approximately 120-140 KIAS)
- [ ] Rotate - 140-150 KIAS (adjust for weight/config)
- [ ] Rotation rate - Smooth 10-15° nose up
- [ ] Climb attitude - Establish approximately 12-15° nose up
- [ ] Positive rate of climb - Verify
- [ ] Landing gear - UP
- [ ] Landing gear lights - Verify extinguished
- [ ] Climb speed - 350 KIAS or as briefed
- [ ] Trim - Adjust to maintain attitude

## Afterburner Takeoff

- [ ] Brakes - Hold firmly
- [ ] Throttle - MIL power
- [ ] Engine stabilized - Check
- [ ] Afterburner - Engage (outboard/inboard)
- [ ] RPM increase - Verify
- [ ] FTIT increase - Monitor (within AB limits)
- [ ] Nozzle - Check opening
- [ ] AB lights - Check illuminated
- [ ] Brakes - Release
- [ ] Acceleration - Monitor rapid increase
- [ ] Rotate - As appropriate for weight (typically 140-160 KIAS)
- [ ] Climb - Establish positive rate
- [ ] Landing gear - UP
- [ ] Afterburner - Cancel at safe altitude or as briefed
- [ ] Throttle - Reduce to MIL or climb power
- [ ] Speed - 350 KIAS / 0.9 Mach in climb

## Heavy Weight / Combat Configuration Takeoff

- [ ] Rotation speed - Increased (160-180 KIAS depending on weight)
- [ ] Rotation rate - More gradual
- [ ] Angle of attack - Monitor (not exceed optimum)
- [ ] Climb gradient - Accept shallower initial climb
- [ ] Gear up - After positive rate and sufficient airspeed
- [ ] Flap retraction - As appropriate for configuration
- [ ] Climb schedule - As briefed for weight

## Crosswind Takeoff

- [ ] Into wind aileron - Apply at start of roll
- [ ] Rudder - Aggressive use for centerline
- [ ] Rotation - Normal speed but be smooth
- [ ] Wings level - Maintain with aileron
- [ ] Drift correction - Apply immediately after liftoff
- [ ] Crab angle - Establish as needed

## Short Field / Maximum Performance Takeoff

- [ ] Brakes - Hold maximum
- [ ] Throttle - Full MIL or AB
- [ ] Engine parameters - Verify max performance
- [ ] Brakes - Release when stabilized
- [ ] Acceleration - Monitor
- [ ] Rotate - At computed V_R (may be higher)
- [ ] Angle of attack - Optimize for best climb gradient
- [ ] Obstacles - Clear with appropriate margin
- [ ] Gear up - Immediately after positive rate
- [ ] Flaps - Retract per schedule
- [ ] Speed - Climb at V_Y (best rate) or V_X (best angle)

## After Takeoff (Initial Climb)

- [ ] Positive rate - Confirmed
- [ ] Landing gear - UP (verify handle and lights)
- [ ] Landing gear transit lights - Verify extinguished
- [ ] Leading edge flaps - AUTO (verify retracted in clean config)
- [ ] Nosewheel steering - Disengaged (automatic)
- [ ] Trim - Adjust for hands-off flight
- [ ] Climb speed - Establish 350 KIAS / 0.9 Mach
- [ ] Throttle - Set climb power (MIL or less as required)
- [ ] Engine instruments - Monitor
  - [ ] FTIT - Within normal range
  - [ ] RPM - As set
  - [ ] Fuel flow - Normal for power setting
  - [ ] Oil pressure - Normal
  - [ ] Hydraulics - Normal (A&B systems)
- [ ] Departure frequency - Contact ATC
- [ ] Departure procedure - Follow as cleared/briefed
- [ ] TCAS - Monitor traffic
- [ ] Altitude restrictions - Comply
- [ ] Speed restrictions - Comply (typically 250 KIAS below 10,000')
- [ ] Configuration changes - Make as required

## Rejected Takeoff (If Necessary)

- [ ] Throttle - IDLE immediately
- [ ] Brakes - Maximum (anti-skid will modulate)
- [ ] Drag chute - Deploy if installed and required
- [ ] Nosewheel - Lower smoothly
- [ ] Speedbrake - Extend
- [ ] Steering - Maintain centerline
- [ ] Emergency ground crew - Notify as required
- [ ] After full stop - Assess situation, make radio calls
