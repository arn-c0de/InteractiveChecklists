# F-16C Viper Landing Checklist

## Descend/Before Landing

- [ ] **FUEL** - CHECK
- [ ] **PROBE HEAT** - OFF
- [ ] **LANDING LGT** - CHECK
- [ ] **ECM PWR** - OFF
- [ ] **ALTIMETER** - SET

## Flight Control Check

- [ ] **FLIGHT CONTROLS** - CYCLE & CHECK

## Landing Actions and Considerations

- [ ] Establish "On AoA" (Angle of Attack) speed by attitude and trim for it.
- [ ] Set rate of descent via power.
- [ ] Get on the glideslope.
- [ ] Flare.
- [ ] Landing speed is typically around 150 to 160 knots.
- [ ] Land on the rear wheels, then apply full speedbrake and hold the nose up until it falls on its own.
