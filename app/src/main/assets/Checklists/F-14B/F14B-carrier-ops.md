# F-14B Tomcat Carrier Operations Checklist

> Carrier launch/landing checklist; for simulation/training only. Not a substitute for official carrier NATOPS or ship procedures.

## Carrier Pre-Launch (Before Taxi)

- [ ] Briefing - Complete, including launch interval, case, and weather
- [ ] Deck crew signals - Briefed and confirmed
- [ ] External Lighting - As required by deck/ship
- [ ] Canopy - Locked and latched
- [ ] Ejection Seats - Armed (as directed)
- [ ] Harness/Leg restraints - Secure
- [ ] Ballast/Weight & Balance - Confirm
- [ ] Fuel - Check sufficient for launch, recovery or divert
- [ ] Master Arm - SAFE (until cleared)
- [ ] Flaps/Slats - Set for catapult launch as required
- [ ] Speedbrake - CHECK RETRACTED
- [ ] Trim - Set for takeoff
- [ ] Radios - Ship/Launch frequency tuned
- [ ] TACAN/ILS - Set and check (if required)

## Catapult Launch Setup

- [ ] Launch Bar - DOWN/ARMED
- [ ] Tailhook - STOWED/UP for launch; confirm pinned/stowed and "HOOK UP" indicator on (or handle in the UP position)
- [ ] Parking Brake - SET until on catapult
- [ ] Flaps - Set per catapult config
- [ ] Stabilators - Set per launch profile
- [ ] Nacelle/Wing Sweep - Set for takeoff
- [ ] Engine instruments - Both engines within limits
- [ ] HUD - Set and check for takeoff data
- [ ] BRIEF - Catapult signals and abort actions confirmed

## Hook/Barrier/Clearance (For Recovery/Touch-and-Go)

- [ ] Briefing - Recovery brief complete
- [ ] Approach speed - Compute and verify
- [ ] Target approach speed - ~125-145 KIAS (approx; weight/config dependent)
- [ ] Final approach/touchdown target - ~120-140 KIAS (approx; arrested landing)
- [ ] Hook lever - DOWN/ARM for landing; verify "HOOK DOWN" indicator and lever position
- [ ] Flaps - Configure for landing (as required)
- [ ] Speed Brake - CHECK RETRACTED
- [ ] Landing Gear - DOWN and locked
- [ ] Landing Lights/NAV - ON as required
- [ ] Tailhook - DOWN/ARMD and checked
- [ ] LSO/Ball calls - Monitor and comply; wave-off on LSO call
- [ ] Wave-off criteria - Not stabilized, out of lineup, or unsafe approach; execute wave-off immediately (bolter)
- [ ] CRT/ICS - Check for final approach instructions
- [ ] Master Arm - SAFE

## After Landing / Arrested Recovery

- [ ] Canopy - Open when safe
- [ ] Ejection seat - SAFE (as required)
- [ ] Chocks/Deck crew - Follow ship signals
- [ ] Speedbrake - As required
- [ ] Flaps - As required
- [ ] Engine bleed / External power (ship/GPU) - As required for shutdown (no APU on F-14B)
- [ ] Secure - Hand signals to deck crew confirmed

---

Notes:
- Follow the ship's approach, launch and deck handling SOPs at all times.
- For catapult launch procedures, ensure catapult bridle and holdback pin checks are performed by deck crew; crew may confirm visually per ship SOP.
- This checklist is a simplified guide for simulation and training only.
