# F-14B Tomcat Maneuvers Checklist

> Tactical and maneuver checks for simulation/training. Not a substitute for official tactics or operational procedures.

## Pre-Maneuver Setup

- [ ] Briefing - Maneuver plan, altitudes, turn points, abort criteria
- [ ] Fuel - Check required reserves for maneuvering and recovery
- [ ] Master Arm - Set per mission (ARM for live, SAFE for training)
- [ ] Flaps - Up (unless maneuver requires different config)
- [ ] Speedbrake - Retracted
- [ ] Trim - Set for maneuvering
- [ ] Hydraulic/Flight Controls - Check pressure/response
- [ ] Engine instruments - Check both engines normal
- [ ] RWR/ECM - Set and monitor
- [ ] Chaff/Flare (CMDS) - Armed or SAFE per mission
- [ ] Radios/Formation ICS - Check
- [ ] Minimum maneuver altitude - Set per mission and safety: recommended >=1000 ft AGL for basic ACM, >=3000 ft AGL for high-energy or formation maneuvers (adjust to mission and restrictions)

## Air Combat Maneuvering (ACM) Basic

- [ ] Radar - Set search/track mode and range
- [ ] HUD - Set for target and engagement
- [ ] BFM - Set energy/angles for the selected tactic (e.g., offensive, defensive)
- [ ] All sensors - Check status
- [ ] Visual clearance - Confirm airspace and formation positions
- [ ] Afterburner - Use as required (monitor temps and fuel)
- [ ] AoA - Monitor for critical limits
- [ ] G-limits - Monitor for structural limits

## High-G / Twist Maneuvers

- [ ] G-suit - On and functioning
- [ ] Oxygen - Flow and pressure nominal
- [ ] Check fuel boost pumps - Set as required
- [ ] Check trim - Maintain proper trim for maneuver
- [ ] Stabilator - Monitor position/response

## Formation / Defensive Maneuvers

- [ ] Formation position - Confirm station/breakout plan
- [ ] Radios/ICS - Confirm calls and hails
- [ ] Use of ECM/RWR - As required for threat environment
- [ ] Countermeasures - Prepare as needed

## Recovery from Maneuver

- [ ] Stabilize aircraft - Set desired power and configuration
- [ ] Trim - Recenter and set for cruise/climb
- [ ] Check systems - Hydraulics, fuel balancing
- [ ] Radios - Confirm return-to-base or recovery procedures
- [ ] Weapons - Safe or configured per mission/landing

---

Notes:
- This checklist avoids specific weapon employment or classified tactics.
- Tailor the maneuvers checklist to your mission profile and add carrier-specific transition checks when integrating with carrier ops.
