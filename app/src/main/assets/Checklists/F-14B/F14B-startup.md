# F-14B Tomcat Startup Checklist

> Note: Reference-style checklist; for simulation or training only. Not a replacement for official procedures.

## Exterior Preflight

- [ ] Overall aircraft condition - Inspect
- [ ] Tires and landing gear - Check condition
- [ ] Intake/engine inlet FOD - Clear
- [ ] Pitot/static covers - Removed
- [ ] Wing fold/latch - Locked (if wings were folded)
- [ ] Wing sweep actuator - Check locked
- [ ] Arresting hook - Stowed/locked
- [ ] External lighting (NAV/Taxi/Formation/Strobe/Landing) - Off/As required
- [ ] Fuel tank caps/doors - Secure
- [ ] Pylons and stores - Inspect as loaded
- [ ] Canopy - Check seals and secure

## Cockpit Entry

- [ ] Ejection seat safety pins - Installed during entry
- [ ] Kneeboard and publications - Stow
- [ ] Loose articles - Remove or secure
- [ ] Seat position/height - Adjust
- [ ] Rudder pedals - Adjust
- [ ] Harness - Connect and lock
- [ ] Leg restraints - Connect
- [ ] Helmet + oxygen mask - Don

## Before Battery Power

- [ ] Circuit breakers - Check IN
- [ ] Emergency Mode switches - Check OFF/Normal
- [ ] Landing Gear handle (LDG GEAR) - DOWN
- [ ] Arresting Hook lever (HOOK) - UP/Stowed
- [ ] WING SWEEP handle - CHECK LOCKED/IN AUTO
- [ ] Launch Bar (if applicable) - SAFE
- [ ] Speedbrake handle - CENTER
- [ ] Flaps - UP or MANUAL as required
- [ ] Exterior lighting - OFF
- [ ] MASTER ARM - SAFE

## Battery Power On

- [ ] BAT switches (Battery/Main) - ON
- [ ] Standby attitude indicator - Check erect
- [ ] Standby compass - Check
- [ ] Cockpit lighting - Set for startup
- [ ] Instrument panel lights - Check

## External Power / Ground Start 

- [ ] External Power (GPU) - Connect/Avail - ON (if required)
- [ ] Air/Start Cart - Connect/Avail - ON (if external air start required)
- [ ] Check GPU/Cart indications - Ensure stable power/air pressure

## Avionics Power Up

- [ ] GEN switches (L GEN / R GEN) - ON after engine start
- [ ] BUS TIE or Bus Transfer - AUTO as required
- [ ] Avionics Master/Avionics Power - ON
- [ ] ICS (Interphone) - Check operation
- [ ] UHF/VHF/HF radios - Set/check frequencies
- [ ] TACAN/ADF - Set if required
- [ ] INS/IRS alignment - Set to NAV, monitor
- [ ] HUD - Power/brightness - Check
- [ ] Radar - Power ON and BITs (if required)
- [ ] RWR - ON/Check
- [ ] CMDS (Countermeasure system) - Check as required

## Before Engine Start

- [ ] Fire bottles - Checked/Armed (as required)
- [ ] Oxygen - Set/flow and pressure 100%
- [ ] G-suit - Connected/Checked
- [ ] Canopy - Close and lock
- [ ] Ejection seat - SAFE / Pins removed when ready
- [ ] Seat - ARM (when ready for flight)
- [ ] Parking Brake - SET
- [ ] Throttles - OFF
- [ ] Fuel pumps - ON (as required by fuel config)
- [ ] Hydraulic pumps - Check/shutdown state
- [ ] Hydraulic pressure - Check

## Engine Start (Right Engine First)

- [ ] RIGHT ENG START switch - SELECT/START
- [ ] Right engine N2 (or Ng/Np)/N1 - Monitor rotation
- [ ] Oil pressure - Check rising (indicates start)
- [ ] EGT/TGT - Monitor limits
- [ ] Right engine - Stabilize at idle N1/N2
- [ ] R GEN - Check online (or generator switch when available)
- [ ] Right engine instruments - NORMAL

## Engine Start (Left Engine)

- [ ] LEFT ENG START switch - SELECT/START
- [ ] Left engine N2 (or Ng/Np)/N1 - Monitor rotation
- [ ] Oil pressure - Check rising
- [ ] EGT/TGT - Monitor limits
- [ ] Left engine - Stabilize at idle N1/N2
- [ ] L GEN - Check online
- [ ] Left engine instruments - NORMAL

## After Engine Start

- [ ] External Power/GPU - Disconnect (if used)
- [ ] Air/Start Cart - Stow/Disconnect (if used)
- [ ] Pack/Environmental control - AUTO
- [ ] Probe heat/Engine anti-ice - As required
- [ ] Cabin press - Check normal
- [ ] INS/IRS - Monitor alignment/progression
- [ ] Flight controls - Check freedom and correct response
- [ ] Trim - Set neutral/check
- [ ] Hydraulic System - Pressure normal
- [ ] Fuel Quantity - Check and balance feeds
- [ ] Stores/Ordnance - Verify status
- [ ] MASTER ARM - SAFE for taxi

## Flight Control Check

- [ ] Flight controls - Full deflection both directions
- [ ] Ailerons - Check deflection
- [ ] Elevators - Check deflection
- [ ] Rudder - Check movement
- [ ] Flaps - Set for taxi/takeoff as required
- [ ] Leading edge slats or flaps (if applicable) - Verify
- [ ] Tab/System - Check
- [ ] Autopilot/AFCS - Off during preflight checks, test if required

## Before Taxi

- [ ] Parking Brake - RELEASE
- [ ] Nosewheel Steering - Check ON/functional (if installed)
- [ ] Brakes - Test & set for taxi
- [ ] Taxi Lights - ON
- [ ] Nav Lights - As required
- [ ] Anti-Collision/Strobe - As required
- [ ] Canopy - Confirm locked and sealed
- [ ] INS - Alignment complete
- [ ] Radios - Set for ground/tower
- [ ] Clearance - Obtain taxi clearance

## Taxi

- [ ] Throttles - Advance smoothly and keep idle as required
- [ ] Nosewheel Steering - Check responsiveness
- [ ] Brakes - Check effectiveness
- [ ] Engine instruments - Monitor both engines
- [ ] Flight controls - Check free and no binding
- [ ] Trim - Confirm takeoff trim
- [ ] Taxi Speed - Maintain safe taxi speed

## Before Takeoff / Line Up

- [ ] Flight Controls - Full and free
- [ ] Trim - Set for takeoff
- [ ] Flaps - Set for takeoff (as required)
- [ ] Speedbrake - CHECK RETRACTED
- [ ] Fuel - Confirm feed and quantity
- [ ] Weight & Balance - Confirm
- [ ] Master Arm - ARM if clearance and required
- [ ] HUD/Radar - Configure for departure
- [ ] Canopy - Locked and latched
- [ ] Harness - Tight

## Takeoff

- [ ] Parking Brake - RELEASE
- [ ] Throttles - Advance smoothly to takeoff power
- [ ] Engine instruments - Monitor both engines
- [ ] Directional control - Maintain centerline
- [ ] Rotate - At calculated Vr (approx 150-170 KIAS, weight/config dependent)
- [ ] Climb - Positive rate, gear up
- [ ] Initial climb speed - Establish and maintain ~200-260 KIAS (approx; adjust to mission/weight)

## After Takeoff / Climb

- [ ] Landing gear - UP and locked
- [ ] Flaps - Retract as required
- [ ] Climb power - Set
- [ ] Trim - Adjust as required
- [ ] Check engine instruments - Normal
- [ ] Departure - Execute as cleared
- [ ] Radios - Contact departure
