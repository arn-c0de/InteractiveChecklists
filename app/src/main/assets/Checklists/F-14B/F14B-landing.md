# F-14B Tomcat Landing Checklist

> Landing checklist covering shore and carrier operations; simulation/training only. Always use official NATOPS/OEM procedures for real operations.

## Approach Preparation (Common)

- [ ] Briefing - Approach profile, landing weight, fuel state, runway or carrier case, weather
- [ ] Radios - Approach/tower/ship frequency set and call confirmed
- [ ] TACAN/ILS - Set and check (if required)
- [ ] HUD - Set landing/approach reference
- [ ] Altimeters - Set QNH / ship reference
- [ ] Trim - Set for approach
- [ ] Flaps - Set to approach config (as required)
- [ ] Speedbrake - Retract/As required
- [ ] Master Arm - SAFE
- [ ] Landing lights - ON

## Field/Shore Landing

- [ ] Approach speed - Target approach speed: approx 120–150 KIAS (adjust for weight/config)
- [ ] Final approach speed - Maintain smooth stabilized approach: approx 120–140 KIAS
- [ ] Glideslope/Descent - Stabilize descent rate and approach path
- [ ] Landing Gear - DOWN and locked (on final)
- [ ] Flaps - Set for landing as required
- [ ] Speedbrakes - Retracted (unless required)
- [ ] Autopilot/AFCS - Disengage/Confirm pilot control on final
- [ ] Fuel - Crossfeed or as required for landing configuration
- [ ] Check engine instruments - Monitor both engines
- [ ] Flare - Execute at safe height; reduce sinkrate for touchdown
- [ ] Touchdown - Maintain centerline and directional control
- [ ] Brakes - Apply as required; deploy speedbrake if needed
- [ ] Taxi - Clear runway and comply with tower instructions

## Carrier Landing (Field arrestor ops in F-14B context)

- [ ] Recovery Brief - Confirm case, approach pattern, landing interval, and marshal
- [ ] Approach speed (carrier) - Target approach speed: ~125–145 KIAS (approx; weight/config dependent)
- [ ] Final approach - Stabilized glidepath to deck (3°–4° typical beam/LSO glideslope references); check for correct deck lineup
- [ ] Hook lever - DOWN/ARM for landing; confirm "HOOK DOWN" indicator light and lever position
- [ ] Flaps - Configure per carrier landing profile (max landing flaps)
- [ ] Landing Gear - DOWN and locked
- [ ] Landing Lights - ON, formation and ID lights as required
- [ ] Tailhook - DOWN/ARM and check; verify hook position if visual confirmation is possible
- [ ] Trim - Set for approach speed
- [ ] LSO/Ball calls - Monitor and obey; execute wave-off on LSO call
- [ ] Wave-off (bolter) - Execute immediately if not stabilized by 500 ft or if LSO calls wave-off: add power, fly positive climb, carry out bolter recovery
- [ ] G limits/Case checks - Manage energy and be prepared for bolter or wave-off
- [ ] Arrested landing - Touchdown inside the ramp at correct sink rate; maintain direction
- [ ] After touchdown - Check hook engagement, follow deck crew signals, taxi to park per ship SOP; inspect tailhook for damage before next launch

## Missed Approach / Bolter

- [ ] Missed approach - Power up, climb and rejoin per recovery brief
- [ ] Bolter (carrier) - Full power, accelerate and fly positive climb, then re-enter pattern
- [ ] Check flaps/nacelle/wings as required and confirm approach configuration for next pattern

## After Landing / Taxi & Shut Down

- [ ] Speedbrake - Set as required
- [ ] Flaps - Set to taxi or as required
- [ ] Brakes - Check and set parking brake once clear
- [ ] Canopy - Open when safe or follow ship procedures
- [ ] Ejection seat - SAFE (if needed)
- [ ] Ground/Deck signals - Follow instructions
- [ ] APU/External power - Configure as required
- [ ] SALUTE/Secure aircraft - Perform post-flight checks

---

Notes:
- Speed guidance is approximate: exact numbers vary with weight, fuel, stores, flaps, and NATOPS tables.
- Carrier approach speeds vary with gross weight and tactical configuration — consult NATOPS or ship SOP for final figures.
- This checklist is a simulation/training reference; use official procedures for operational use.
